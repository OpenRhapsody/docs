# 광고 상품 만들기

## ⚡ 퀵스타트: 첫 광고 상품 만들기
- 🎬 데모 영상: 광고 상품 생성부터 판매 시작까지
- ⏱️ 소요 시간: 약 5분
- 📋 시작 전 준비:
  - 유닛 ID (광고 유닛 설정에서 생성)
  - 상품명, 가격 정책, 단가
- ✅ 완료 후: 광고 상품 판매 시작
- 👉 다음 단계: 타겟팅 카테고리 설정

## 광고 상품이란?
- 개념: 하나 이상의 광고 유닛을 묶어 판매 단위로 구성한 묶음
- 광고 상품과 유닛의 관계
- 3단계 프로세스: 앱 생성 → 광고 유닛 생성 → 광고 상품 생성

## 광고 상품 생성하기
- 상품명 작성 가이드
  - 권장 네이밍 컨벤션: `[플랫폼] + [위치] + [포맷] + [특징]`
  - 좋은 예시 / 피해야 할 예시
- 과금 유형 (Pricing Type)
  - CPM: 1,000회 노출당 과금
  - CPC: 클릭당 과금
  - CPP: 기간/패키지당 과금
    - 슬롯 수 (cppSlots): 동시 노출 가능한 광고 수
    - 최소 기간 (cppMinDays): 최소 구매 기간
  - 동영상 프리미엄 (Video Addon)
    - 동영상 에드온 추가 단가 설정
    - 스킵 지연 시간 설정 (사용자가 스킵 가능해지기까지 대기 시간)
- 판매 정보 (Sales Information)
  - 표시 통화 (Display Currency)
  - 시간대 (Timezone)
  - 단가 (Unit Price)
- 예상 성과 지표
  - 일일 노출수 (Daily Impressions)
  - 평균 CTR
  - 인벤토리 예약: 일별 가용량 계산에 활용
- 소재 가이드 링크 (선택)
- 타겟팅 카테고리 연결 (상세: 타겟팅 카테고리 섹션 참조)

## 유닛 세트 설정
- 유닛 세트란?
  - 광고주에게 소재를 제공받을 지면들의 묶음
  - 유닛 세트 이름이 광고주에게 표시됨
- 자동 통합
  - 동일한 사이즈·형식의 유닛은 자동으로 한 세트로 지정
  - 소재가 공유됨
- 통합 관리 vs 분리 관리
- 유닛 세트 옵션
  - 템플릿 사용 여부
  - 워터마크 지원 여부
- 주의사항 (⚠️)
  - 유닛 세트 미지정 시 소재 등록/노출 불가
  - 형식이 같아도 사이즈 다르면 별도 세트
  - 수정 필요 유닛 (fixRequired) 포함 시 캠페인 생성 불가

## 광고 상품 테스트하기
- 🎬 데모 영상: 테스트 모드 설정 및 확인
- 테스트 모드란?
  - 광고 상품 라이브 전 실제 동작 확인
  - 테스트 노출은 과금/통계에서 제외
- 테스터 등록
  - IP 주소 기반 화이트리스트
  - 테스터별 유닛 지정 (특정 유닛만 테스트 가능하도록 제한)
  - 테스트 노출 카운터로 확인
- 테스트 순서
  1. 테스터 IP 주소 등록
  2. 테스트 활성화
  3. 해당 IP에서 앱/웹 접속하여 광고 확인
  4. 테스트 완료 후 비활성화
  5. 광고 상품 라이브
- 테스트 데이터 관리
  - 일정 기간 후 자동 정리 (TTL)

## 사전 점검 캠페인
- 사전 점검 캠페인이란?
  - 광고 상품 판매 전 성과 데이터를 확보하는 절차
  - 실제 캠페인과 동일한 환경에서 진행
- 목적
  - 신뢰도 확보: 실제 성과 데이터(노출수, CTR 등)로 광고주 설득력 향상
  - 성과 검증: 애드컨트롤과 GA 등 외부 분석 도구 간 차이 확인
  - 상품 최적화: 테스트 결과 기반 상품 구성/단가 확정
  - 레퍼런스 확보: 광고주 카테고리별 사례 수집
- 운영 방식
  - CPP 상품으로 진행
  - 최소 1주일 이상 운영 권장
  - 모든 유닛에서 실제 광고 노출

## 광고 상품 수정하기
- 광고 상품 상태
  - 준비중: 테스트 미완료 또는 유닛 미동기화
  - 판매중: 캠페인 예약 가능
  - 삭제중: 삭제 요청 후 완전 삭제 전
- 수정 가능 항목
- 캠페인 영향 (⚠️ 중요)
  - 진행 중인 캠페인에는 영향 없음
  - 수정 사항은 새 캠페인부터 적용

## 광고 상품 삭제하기
- 삭제 조건
- 삭제 후 복구

## 광고 상품 홍보하기
- 광고 프로필 설정
  - 플랫폼 이름, 대표 링크
  - 테마 (앱 이미지, 브랜드 컬러, 썸네일)
  - 연락처 (문의 이메일)
  - 지역 설정 (국가, 시간대, 언어)
- 광고 상품 소개서 등록
  - 서비스 카테고리
  - 한 줄 소개
  - 자세한 설명
- 혜택 정보 등록
  - 할인, 무료 체험, 보너스 프로모션
  - 에이드랍 홈페이지 게시
- 광고 상품 목록 관리하기
  - 노출 순서 변경

## 고급 설정
- 노출 빈도 제한 (Exposure Frequency Cap)
- 할인율 설정
- 글로벌 지원
  - 로케일 설정: 국가 (country), 언어 (language), 지역 (region)
  - 시간대 설정: 캠페인 시작/종료 기준
  - 다통화 설정: 표시 통화 선택
- 비공개 광고 상품
  - 비공개 설정 시 동작
  - 활용 사례
- InHouse 광고 상품 (⚠️ 사용 권장하지 않음)
  - InHouse란?
  - 광고주에게 판매하면 안 되는 이유
  - 노출 정책
  - 옵션별 동작

## 광고 상품 문제해결
- 주의가 필요한 광고 상품
  - 수정 필요 유닛 포함 (adUnitFixRequired)
  - 테스트 미완료 (testRequired)
- 알림 해결 방법
