# 캠페인 수익금 지급

캠페인 종료 후 수익금을 정산받는 프로세스를 안내합니다.

> 💡 수익금을 정산받으려면 사업자 정보와 지급 계좌가 등록되어 있어야 합니다. [수익금 지급](../ad-platform/payout.md)에서 등록 방법을 확인하세요.

---

## 정산이란?

캠페인이 종료되면 광고주가 결제한 금액에서 매체사 몫을 정산받습니다. 정산은 캠페인 종료 기준으로 월별로 진행됩니다.

---

## 정산 일정

| 시점 | 내용 |
|------|------|
| **캠페인 종료 익월 5영업일 이내** | 확정된 수익금 정산서 이메일 발송 |
| **매월 10일** | 이의제기 마감, 세금계산서 발행 마감 |
| **매월 21일** | 수익금 지급 (주말/공휴일 시 익영업일) |

**예시**: 6월 종료 캠페인 → 7월 10일까지 세금계산서 발행 → 7월 21일 수익금 지급

---

## 결제 방식별 프로세스

### 카드 결제 (선결제)

광고주가 캠페인 예약 시 카드로 선결제한 경우입니다. 애드컨트롤이 광고주로부터 결제 금액을 수령합니다.

| 항목 | 내용 |
|------|------|
| **정산서 발행** | 캠페인 종료 익월 5영업일 이내 |
| **수익금 지급** | 정산서 이의제기 없으면 매월 21일 (또는 익영업일) |
| **세금계산서** | 매월 10일까지 발행 필수 |
| **지급 정보 변경** | 매월 10일 이전까지 요청 (20일 이후 변경은 익월 반영) |

> 💡 캠페인별 정산서는 캠페인 상세 페이지에서 확인할 수 있습니다.

### 은행 송금

| 항목 | 내용 |
|------|------|
| **수익금 지급** | 매월 21일 (또는 익영업일) |
| **지급 정보 변경** | 매월 20일 이전까지 완료 필요 |
| **지급 조건** | 누적 잔액이 지급 기준액 충족 시 지급 |
| **이월** | 기준 미달 또는 지급 보류 상태면 익월로 이월 |

### 수동 결제 (후불)

매체사가 광고주와 직접 정산하는 경우입니다.

- 매체사가 광고주에게 직접 청구
- 세금계산서 발행도 직접 협의

---

## 자동 환불

캠페인 종료 후 목표 대비 성과가 미달하면 차액이 자동 환불됩니다.

| 조건 | 설명 |
|------|------|
| **대상** | CPM, CPC 캠페인 |
| **결제 방식** | 카드 결제만 해당 |
| **처리** | 환불액은 정산 금액에서 차감 |

> 💡 수동 결제 캠페인은 매체사와 광고주가 직접 협의하여 처리합니다.

## FAQ

**Q. 정산서에 이의가 있으면 어떻게 하나요?**

매월 10일까지 정산 주 이메일로 회신하여 이의를 제기하세요.

**Q. 21일이 주말이면 언제 지급되나요?**

익영업일에 지급됩니다.

**Q. 수동 결제 캠페인도 정산서가 발송되나요?**

네. 정산서는 발송되지만, 실제 정산은 광고주와 직접 진행해야 합니다.
