# 캠페인 관리하기

## ⚡ 퀵스타트: 캠페인 심사하기
- 🎬 데모 영상: 캠페인 심사부터 승인까지
- ⏱️ 소요 시간: 약 3분
- 📋 시작 전 준비:
  - 심사 대기 중인 캠페인
- ✅ 완료 후: 캠페인 노출 시작
- 👉 다음 단계: 성과 모니터링

## 캠페인 현황 조회

### 캠페인 목록
- 예약 관리 메뉴에서 전체 캠페인 조회
- 필터링 옵션
  - 캠페인 상태
  - 과금 유형
  - 광고 상품
  - 캠페인 기간

### 조회 방식
- 리스트 뷰
  - 표 형태로 캠페인 비교
  - 상태, 캠페인명, 소진 비용/예산, 기간, 성과 표시
- 타임라인 뷰
  - 일정 형태의 막대로 표현
  - 동일 기간 캠페인 비교에 용이

### 캠페인 상태
- 작성중: 광고주가 작성 중 (콘솔에 미표시)
- 심사중: 매체사 검토 대기
- 거절됨: 심사 거절
- 예약됨: 심사 승인, 시작일 대기
- 진행중: 광고 노출 중
- 정지됨: 일시정지 상태
- 취소됨: 캠페인 취소
- 종료됨: 기간 만료

## 캠페인 심사하기

### 캠페인 승인/거절
- 심사 필요 캠페인 확인
  - '심사중' 상태로 필터링
- 캠페인 정보 검토
  - 광고 상품, 기간, 예산, 타겟팅 확인
- 승인 처리
  - 승인 시 예약됨 상태로 변경
  - 시작일에 자동으로 진행중 상태로 전환
- 거부 처리
  - 거절 사유 작성
  - 광고주에게 알림 발송

### 심사 기한 (⚠️ 중요)
- 다음 중 먼저 도래하는 시점까지 심사 완료 필요:
  - 캠페인 예약 후 72시간 이내
  - 캠페인 시작 전
- 심사 기한 초과 시 자동 거절

### 당일 예약 캠페인
- 당일 시작 캠페인은 특별 규칙 적용
- 캠페인 시작 후 24시간 내 미승인 시 자동 거절

### 소재 심사하기
- 심사 필요 소재 확인
  - '소재 심사 필요' 뱃지가 붙은 캠페인
- 심사 항목
  - 소재 이미지/동영상 확인
  - 도착 URL 확인 (소재 클릭 시 이동 경로)
- 승인/거부 처리
  - 승인: 광고 송출 가능
  - 거부: 거절 사유 선택 후 반려
- 거절 사유 종류
  - 법률 위반 (law)
  - 허위/과장 광고 (fake)
  - 비방/명예훼손 (slander)
  - 음란물 (obscene)
  - 폭력성 (violence)
  - 저작권/초상권 침해 (right)
  - 사회 규범 위반 (norm)
  - 청소년 유해 (teenager)
  - 정책 위반 (policy)
  - 허용되지 않는 콘텐츠 (disallowed)
  - 소재 오류 (asset_failed)
  - 기타 (direct)
- 주의사항 (⚠️)
  - 캠페인이 승인되어도 승인된 소재가 없으면 광고 송출 안 됨
  - 거절 시 광고주는 수정 후 재제출 가능

## 캠페인 운영하기

### 캠페인 상세 정보
- 캠페인 ID
  - 자동 부여되는 고유 식별자
- 예산 소진
  - 설정 기간/예산 기준 소진 금액
  - 진행률 표시
- 전체 성과
  - 노출 수, 클릭 수 요약
- 광고 소재
  - 현재 노출 중인 소재 목록
  - 소재별 노출 상태
- 일 할당량 자동 조정
  - CPM/CPC 캠페인은 매일 자정에 남은 목표를 기준으로 일 할당량 자동 재계산
  - 캠페인 초반에 약간 더 많은 할당량 배정 (균등 분배 대비)

### 캠페인 수정하기
- 수정 가능 항목
  - 광고 기간 (진행중/정지 상태에서는 시작일 수정 불가)
  - 타겟팅
  - 캠페인 이름
  - 노출 빈도 제한
  - 일 클릭/노출 할당량
- 수정 불가능 항목 (⚠️)
  - 광고 상품
  - 예산
  - 결제 정보

### 캠페인 일시정지하기
- 일시정지 효과
  - 광고 송출 중단
  - 성과 집계 중단
- 재시작
  - 더보기(⋮) 메뉴 → 캠페인 재시작
- 소재별 일시정지
  - 개별 소재만 노출 중지 가능
- 주의사항 (⚠️)
  - 광고주 동의 없이 중단 시 분쟁 발생 가능
  - 충분한 소통 후 진행 권장

### 캠페인 취소하기
- 취소 절차
  - 더보기(⋮) 메뉴 → 캠페인 취소
  - 최종 정산 금액(매체사 수취액) 입력 필수
- 결제 유형별 처리
  - 자동결제: 최종 정산 금액 기준 정산
  - 선결제: 선결제 금액 - 최종 정산 금액 = 부분 환불
- 전액 환불
  - 최종 정산 금액을 0원으로 설정
- 자동 환불 처리
  - 캠페인 종료 후 3시간 이상 경과 시 자동 처리
  - CPM/CPC 캠페인, 카드 결제만 해당
  - 목표 대비 성과 미달 시 차액 환불
- 주의사항 (⚠️)
  - 광고주 동의 없이 취소 시 분쟁 발생 가능
  - 일시적 중단만 필요하면 일시정지 사용 권장

### 캠페인 복제하기
- 복제 방법
  - 캠페인 상세 → 더보기(⋮) → 캠페인 복제
- 복제 시 시작일
  - 현재로부터 7일 후로 자동 설정
- 복제 제약 조건 (⚠️)
  - 광고 상품에 유닛이 삭제된 경우 복제 불가
  - 유닛 세트에 수정 필요한 경우 복제 불가
- 복제 후 상태
  - 콘솔에서 복제: 작성중 상태, 광고 계정에 표시
- 최종 확정
  - 약관 동의 및 최종 제출 → 심사중 상태로 변경

## 매체사 직접 예약

### 직접 예약이란?
- 매체사가 광고주 대신 캠페인 생성
- 수동 결제만 가능
- 광고주와 사전 협의 필수

### 캠페인 생성하기
- 생성 위치
  - 예약 관리 → 캠페인 생성 버튼
- 광고 계정 선택
  - 기존 광고 계정 선택 또는 새로 생성
- 캠페인 정보 입력
  - 광고 상품 선택
  - 시작일/종료일 지정
  - 캠페인 이름, 예산 설정
- 결제 정보 입력
  - 담당자 정보
  - 결제 수단 (수동 결제)
  - 청구 정보
- 주의사항 (⚠️)
  - 시작일이 지난 캠페인은 생성 불가
  - 광고주 동의 없는 임의 생성 시 분쟁 발생 가능

### 소재 등록하기
- 등록 위치
  - 캠페인 상세 → 광고 소재 섹션
- 소재 등록 옵션
  - 템플릿 사용: 에이드랍 제공 템플릿 활용
  - 자유 이미지: 광고주 제공 이미지 사용
- 필수 정보
  - 소재 유형 (유닛 세트별)
  - 소재 이름
  - 도착 URL
  - 도착 위치 (기본 브라우저 / 인앱 브라우저)
  - 광고 이미지 (등록 사이즈의 3배 권장)
- 고급 설정
  - 노출 기간/시간 지정

### 소재 미리보기
- 내 기기에서 보기
  - 실제 기기에서 광고 소재 노출 확인
- 이용 방법
  1. 캠페인 상세에서 소재 클릭
  2. '내 기기에서 보기' 버튼
  3. 기기 정보 입력 (기기명, ADID)
  4. 실제 기기에서 확인
- 지원 환경
  - Android SDK 1.3.1+
  - iOS SDK 1.3.0+
  - Flutter/React Native SDK 1.1.1+
  - 웹 API: 상시 지원

### InHouse 캠페인
- InHouse 캠페인이란?
  - 자체 프로모션용 캠페인
  - 과금/정산 없음
- 생성 방법
  - 일반 캠페인과 동일, InHouse 옵션 선택
- 노출 우선순위
  - 유료 광고보다 낮은 순위
  - 옵션으로 동일 순위 설정 가능

## 알림

### 캠페인 시작/종료 알림
- 캠페인 시작 1시간 전 알림 발송
- 캠페인 종료 1시간 전 알림 발송

### 소재 액션 필요 알림
- 캠페인 시작 7일 전부터 알림
- 소재 미등록/거절 시 광고주에게 알림
- 소재 심사 대기 시 매체사에게 알림

### 성과 미달 알림
- 캠페인 시작 후 24시간 이상 경과한 경우
- 일일 목표 미달성 시 매체사에게 알림

## 운영 설정

### 캠페인 예약 약관
- PDF 형식 약관 업로드
- 광고주가 캠페인 예약 시 동의

### 결제 방식
- 카드 결제
  - 선결제, 자동 정산
  - 매체사는 카드 결제 캠페인 직접 생성 불가 (⚠️)
- 수동 결제
  - 매체사-광고주 직접 정산

### 광고주 권한 설정
- 권한 종류
  - 당일 예약 허용 여부
  - 수동 결제 허용 여부
  - InHouse 캠페인 허용 여부
- 광고주별 개별 설정 가능

### 광고주 허용 목록
- 이메일/도메인별 권한 설정
- 비공개 상품 접근 제어

### 소재 노출 스케줄 허용
- 광고주가 요일/시간별 노출 설정 가능하도록 허용

### CPP 예산/타겟팅 수정 허용
- 허용된 광고주만 CPP 예산 및 타겟팅 직접 설정 가능
