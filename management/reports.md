# 캠페인 리포트

캠페인 성과를 확인하고 분석하는 방법을 안내합니다.

> 💡 리포트 기본 사용법은 [분석 및 리포트](../campaign/analytics.md)를 참고하세요. 이 문서에서는 매체사 전용 기능을 설명합니다.

---

## 광고주 리포트 설정

광고주에게 제공되는 리포트의 범위를 설정할 수 있습니다.

> 💡 리포트 화면 우측 상단의 **리포트 설정** 버튼을 클릭하면 설정 모달이 열립니다.

| 항목 | 설명 |
|------|------|
| **캠페인** | 캠페인 탭 표시 (기본 활성화, 변경 불가) |
| **소재** | 소재 탭 표시 (기본 활성화, 변경 불가) |
| **광고 유닛** | 유닛 탭 표시 여부 |
| **광고 상품** | 상품 탭 표시 여부 |
| **날짜** | 일별 분석 표시 (기본 활성화, 변경 불가) |
| **시간** | 시간별 분석 표시 여부 |

- **활성화**: 광고주가 해당 탭 또는 시간 단위 분석을 사용할 수 있습니다
- **비활성화**: 광고주 리포트에서 해당 기능이 숨겨집니다

## Backfill 탭

Backfill 광고 수익을 확인할 수 있는 매체사 전용 탭입니다.

### 지표

| 지표 | 설명 |
|------|------|
| **노출수** | Backfill 광고 노출 횟수 |
| **노출률** | Backfill 광고 노출률 |
| **클릭수** | Backfill 광고 클릭 횟수 |
| **CTR** | 클릭률 |
| **CPC** | 클릭당 비용 |
| **CPM** | 1,000회 노출당 비용 |
| **수익** | Backfill 광고 수익 |

### 세그먼트

유닛 세그먼트를 선택하면 광고 유닛별 Backfill 수익을 확인할 수 있습니다.

### 제한사항

Backfill 탭에서는 다음 기능을 사용할 수 없습니다.

- 일별/시간별 분석
- 필터
- 정렬

## 매체사 전용 컬럼

다음 컬럼은 매체사 리포트에만 표시되며, 광고주에게는 표시되지 않습니다.

| 컬럼 | 설명 |
|------|------|
| **비공개 캠페인** | 비공개 상품으로 예약된 캠페인 여부 |
| **애드노트 캠페인** | 애드노트를 통해 예약된 캠페인 여부 |
| **수수료** | 대행사 수수료 금액 |
| **정산** | 최종 정산 금액 (예산 - 수수료 - 환불) |

## FAQ

**Q. 광고주가 Backfill 탭을 볼 수 있나요?**

아니요. Backfill 탭은 매체사 전용입니다. 광고주 리포트에는 표시되지 않습니다.

**Q. 광고주 리포트 설정을 변경하면 언제 반영되나요?**

즉시 반영됩니다. 설정 변경 후 광고주가 리포트를 조회하면 새로운 설정이 적용됩니다.

**Q. 정산 컬럼의 금액은 어떻게 계산되나요?**

예산에서 대행사 수수료와 환불 금액을 제외한 최종 정산 금액입니다.
