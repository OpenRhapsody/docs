# 캠페인 심사하기

광고주가 제출한 캠페인과 소재를 검토하고 승인/거절하는 방법을 안내합니다.

> 💡 광고가 송출되려면 캠페인과 소재가 모두 승인되어야 합니다. 캠페인과 소재를 함께 심사하고, 소재가 없으면 광고주에게 업로드를 요청하세요.

---

## 캠페인 목록 조회

좌측 메뉴에서 **예약 관리**를 클릭하면 캠페인 목록을 확인할 수 있습니다.

### 조회 방식

화면 상단의 탭에서 조회 방식을 선택합니다.

- **리스트 뷰**: 표 형태로 캠페인을 비교합니다. 상태, 캠페인명, 소진 비용/예산, 기간, 성과가 한눈에 표시됩니다.
- **타임라인 뷰**: 캘린더 형태로 캠페인 일정을 시각화합니다. 동일 기간에 진행되는 캠페인을 비교하기 좋습니다.

### 필터

상단 필터를 사용하여 원하는 캠페인을 빠르게 찾을 수 있습니다.

| 필터 | 설명 |
|------|------|
| **검색** | 캠페인명으로 검색 |
| **기간** | 월 단위 또는 날짜 범위 선택 |
| **상태** | 심사중, 예약됨, 진행중 등 상태별 필터 |
| **과금 유형** | CPM, CPC, CPP 등 과금 방식별 필터 |
| **광고 상품** | 상품별 필터 |

### 알림 뱃지

캠페인 목록에서 다음 뱃지를 확인하세요.

| 뱃지 | 의미 |
|------|------|
| **캠페인 심사 필요** | 제출된 캠페인이 심사 대기 중입니다. |
| **임박한 예약** | 시작일이 가까운 캠페인입니다. 우선 심사하세요. |
| **소재 심사 필요** | 승인된 캠페인에 심사 대기 중인 소재가 있습니다. |

## 캠페인 심사

> 💡 캠페인 상태에 대한 자세한 설명은 [캠페인 운영하기](../campaign/operation.md)를 참고하세요.

### 심사 대기 캠페인 확인

1. 좌측 메뉴에서 **예약 관리**를 클릭합니다.
2. 상단 필터에서 **심사중** 상태를 선택합니다.
3. 캠페인을 클릭하여 상세 화면을 엽니다.

### 캠페인 상세 화면

캠페인 상세 화면에서 다음 정보를 확인할 수 있습니다.

**기본 정보**

| 항목 | 설명 |
|------|------|
| 광고 상품 | 선택한 광고 상품과 할인율, 예상 CTR |
| 과금 유형 | CPM, CPC 등 과금 방식과 단가 |
| 예산 | 캠페인 총 예산 |
| 캠페인 기간 | 시작일과 종료일 (타임존 포함) |
| 국가 | 광고 노출 국가 |

**광고주 정보**

| 항목 | 설명 |
|------|------|
| 광고주 | 광고주명과 URL |
| 계약 담당자 | 담당자 이름과 이메일 |
| 결제 수단 | 카드 결제(선불) 또는 수동 결제(후불) |
| 청구 정보 | 사업자명, 사업자등록번호, 이메일 |

**타겟팅**

설정된 타겟팅 조건을 확인합니다. 카테고리, 그룹, 트래픽 소스 등이 표시됩니다.

### 승인 처리

1. 캠페인 상세 화면 상단의 심사 패널에서 **캠페인 승인** 버튼을 클릭합니다.
2. 확인 모달에서 **승인** 버튼을 클릭합니다.
3. 캠페인 상태가 **예약됨**으로 변경됩니다.
4. 시작일에 자동으로 **진행중** 상태로 전환됩니다.

> ⚠️ 기간 보장형(CPP) 상품에서 해당 기간의 예약 가능 슬롯이 없으면 승인이 실패합니다. 이 경우 캠페인을 거절하고 광고주에게 일정 조정을 요청하세요.

### 거절 처리

1. 캠페인 상세 화면 상단의 심사 패널에서 **캠페인 거절** 버튼을 클릭합니다.
2. 거절 사유를 선택합니다.
3. **거절** 버튼을 클릭합니다.
4. 광고주에게 거절 알림이 발송됩니다.

### 거절 사유

| 사유 | 설명 |
|------|------|
| 이미 예약된 일정 | 요청한 기간에 다른 캠페인이 이미 예약되어 있습니다. |
| 브랜드 정책 | 매체사의 브랜드 정책에 맞지 않습니다. |

### 거절 시 환불

카드 결제 캠페인이 거절되면 전액 자동 환불됩니다. 수동 결제 캠페인은 결제 전이므로 별도 환불이 필요 없습니다.

### 심사 기한

캠페인은 다음 중 먼저 도래하는 시점까지 심사를 완료해야 합니다.

- 캠페인 제출 후 **72시간 이내**
- **캠페인 시작 전**

> ⚠️ 심사 기한을 초과하면 캠페인이 자동 거절되고 광고주에게 알림이 발송됩니다.

**당일 예약 캠페인**: 캠페인 시작일 이후에 제출된 경우, 시작 후 **24시간 내** 미승인 시 자동 거절됩니다.

## 소재 심사

> 💡 캠페인 운영 중에도 광고주가 소재를 수정하면 재심사가 필요합니다. 수정된 소재는 승인 전까지 노출되지 않습니다.

### 심사 대기 소재 확인

캠페인 목록에서 **소재 심사 필요** 뱃지가 붙은 캠페인을 클릭합니다. 캠페인 상세 화면 상단에 심사 대기 중인 소재 목록이 표시됩니다.

### 소재 검토

1. 심사 대기 소재를 클릭하여 소재 상세 화면을 엽니다.
2. 다음 항목을 확인합니다.
   - 소재 이미지/동영상 미리보기
   - 도착 URL (소재 클릭 시 이동 경로)
   - 소재 규격 (크기, 포맷)

### 승인/거절 처리

1. 소재 상세 화면에서 **승인** 또는 **거절** 버튼을 클릭합니다.
2. 거절 시 사유를 선택합니다.
3. 광고주에게 심사 결과 알림이 발송됩니다.

거절된 소재는 광고주가 수정 후 다시 제출할 수 있습니다.

### 소재 거절 사유

거절 시 사유를 선택합니다. 현행법 위반, 허위/과장 광고, 음란/선정, 저작권 침해, 광고 불가 업종 등 다양한 사유가 제공됩니다. 해당 사유가 없으면 직접 입력할 수 있습니다.
