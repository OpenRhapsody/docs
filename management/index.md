## 캠페인 현황 조회

### 캠페인 목록

예약 관리 메뉴에서 전체 캠페인을 조회할 수 있습니다.

**필터링 옵션**

| 필터 | 설명 |
|------|------|
| 캠페인 상태 | 심사중, 예약됨, 진행중 등 |
| 과금 유형 | CPM, CPC, CPP |
| 광고 상품 | 특정 광고 상품으로 필터 |
| 캠페인 기간 | 시작일/종료일 범위 |

### 조회 방식

| 뷰 | 설명 |
|------|------|
| **리스트 뷰** | 표 형태로 캠페인 비교. 상태, 캠페인명, 소진 비용/예산, 기간, 성과 표시 |
| **타임라인 뷰** | 일정 형태의 막대로 표현. 동일 기간 캠페인 비교에 용이 |

### 캠페인 상태

| 상태 | 설명 |
|------|------|
| **작성중** | 광고주가 작성 중 (콘솔에 미표시) |
| **심사중** | 매체사 검토 대기 |
| **거절됨** | 심사 거절 |
| **예약됨** | 심사 승인, 시작일 대기 |
| **진행중** | 광고 노출 중 |
| **정지됨** | 일시정지 상태 |
| **취소됨** | 캠페인 취소 |
| **종료됨** | 기간 만료 |

## 알림

### 캠페인 시작/종료 알림

- 캠페인 시작 1시간 전 알림 발송
- 캠페인 종료 1시간 전 알림 발송

### 소재 액션 필요 알림

- 캠페인 시작 7일 전부터 알림
- 소재 미등록/거절 시 광고주에게 알림
- 소재 심사 대기 시 매체사에게 알림

### 성과 미달 알림

- 캠페인 시작 후 24시간 이상 경과한 경우
- 일일 목표 미달성 시 매체사에게 알림
