# 광고 계정

## ⚡ 퀵스타트: 광고 계정 생성하기
- 🎬 데모 영상: 광고 계정 생성
- ⏱️ 소요 시간: 약 3분
- 📋 시작 전 준비:
  - 광고주 정보 (이름, 웹사이트)
- ✅ 완료 후: 광고주가 캠페인 예약 가능
- 👉 다음 단계: 캠페인 심사

## 광고 계정이란?
- 광고주가 캠페인을 예약하고 관리하는 단위
- 매체사가 광고주를 관리하는 메뉴
- 계약 정보, 연락처, 청구 정보 확인 가능

## 광고주 목록

### 계정 정보 조회
- 계정명
- 광고주 웹사이트 링크
- 계약 담당자 연락처 및 이메일

### 청구 정보 조회
- 사업자명
- TAX ID (사업자등록번호)
- 사업자등록증 (아이콘 클릭으로 확인)
- 정산 담당자 이메일

### 광고주 검색 및 필터
- 계정명으로 검색
- 상태별 필터링

## 광고 계정 생성

### 매체사가 직접 생성
- 콘솔에서 [광고 계정 생성] 버튼 클릭
- 광고주 정보 입력
  - 광고주명
  - 웹사이트 링크
  - 국가, 시간대, 언어
- 생성된 계정은 자동으로 광고 플랫폼에 연결

### 광고주가 직접 가입
- 광고 플랫폼 링크 공유: https://{매체사}.adrop.io
- 광고주가 직접 회원가입 후 계정 생성
- 👉 상세 내용: [캠페인 예약하기 > 광고 계정 만들기](./plan-06-campaign-booking.md#광고-계정-만들기) 참조

## 광고주 권한 설정

### 접근 가능 상품 설정
- 광고주별 접근 가능한 광고 상품 지정
- 비공개 상품 접근 권한 부여

### 캠페인 생성 권한
- 당일 예약 허용 여부
- 수동 결제 허용 여부
- InHouse 캠페인 허용 여부

### 결제 방식 설정
- 카드 결제 허용
- 수동 결제 허용
