## 정산 정보 등록

### 청구 정보
- 국가 (광고 프로필 설정 따름)
- 도시
- 사업자명
- TAX ID (사업자등록번호)
- 사업자등록증
- 대표 이메일 (청구 관련 안내 수신)
- 참조 이메일 (쉼표 또는 줄바꿈으로 구분)

### 수익금 지급 계좌
- 예금주명
- 은행명
- 계좌번호
- 통장 사본 (파일 또는 이미지)
- SWIFT Code (BIC) - 해외 송금 시 필수

### 주의사항 (⚠️)
- 정확한 정보 입력 필수
- 오류 시 수익금 지급 지연 가능

## 캠페인 수익금 지급

### 정산이란?
- 캠페인 종료 후 매체사가 받는 광고 수익
- 월별 정산
- 결제 방식에 따라 프로세스 상이

### 정산 일정
- 캠페인 종료 익월 5영업일 이내: 정산서 이메일 발송
- 매월 10일: 이의제기 마감, 세금계산서 발행 마감
- 매월 21일: 수익금 지급 (주말/공휴일 시 익영업일)

### 결제 방식별 프로세스

#### 카드 결제 (선결제)
- 광고주가 캠페인 집행 금액을 카드로 선결제
- 캠페인 종료 후 5영업일 이내 정산서 발송
- 이의 없으면 21일 지급
- 세금계산서는 매월 10일까지 발행
- 지급 정보 변경은 10일 전까지 이메일 요청

#### 계좌이체
- 21일 지급 (주말/공휴일 시 익영업일)
- 지급 정보 변경은 20일 전까지 완료
- 누적 잔액이 지급 기준액 이상이어야 처리
- 미달 또는 보류 상태 시 익월로 이월

#### 수동 결제
- 매체사가 광고주와 직접 정산
- 세금계산서 발행도 직접 협의

### 자동 환불
- 캠페인 종료 후 목표 대비 성과 미달 시 차액 환불
- CPM/CPC 캠페인, 카드 결제만 해당
- 환불액은 정산 금액에서 차감

## 백필광고 수익금 지급

### 백필 수익이란?
- 직광고 미노출 시 백필 광고 네트워크에서 발생한 수익
- 유닛별 수익 집계

### 정산 일정 (M+2 정산)
- 수익 발생월(M) 기준 2개월 후 정산
- 매월 5일: 백필 광고 수익 확정 정산서 이메일 발송
- 매월 10일: 이의제기 마감, 세금계산서 발행 마감
- 매월 21일: 수익금 지급 (주말/공휴일 시 익영업일)
- 예시: 10월 수익 → 12월 10일까지 확정 → 12월 21일 지급

### 환율 적용
- 원화로 지급
- 수익 발생월(M)의 월평균 전신환매도율 적용

### 수익 확인
- 리포트 > Backfill 탭에서 확인
- 노출수, 클릭수, CTR, CPC, CPM, 수익금

## 애드컨트롤 인프라 비용

### 요금제

#### Stream 플랜
- 무료로 매월 1만 노출 제공
- 무료 사용량 초과 시 서비스 제한

#### Wave 플랜
- 무료로 매월 1만 노출 제공
- 초과 시 1천 노출당 280원 (한국)
- 해외: $0.25/CPM

#### 커스텀 플랜
- 별도 문의: biz@adrop.io

### 요금제 미설정 시 (⚠️)
- 광고주에게 "개발중인 서비스" 경고 표시
- Wave 플랜 이상 설정 권장

### 결제 수단

#### 한국 카드 (토스페이먼츠)
- 원화 결제
- 수수료 3.5%

#### 글로벌 카드 (Stripe)
- 다통화 지원
- 수수료 5.5% + 원천징수세 가능

### 청구 주기
- 매월 5일 청구
- 표시 금액은 세전, 최종 청구 시 세금 포함
- 크레딧이 있으면 먼저 차감 후 잔액 청구

### 사용량 확인
- 월별 사용량 및 예상 청구액 매일 정오 업데이트
- 사전 점검 캠페인 크레딧은 청구에서 차감
- PDF 청구서 다운로드 가능

### 환불 정책
- 콘솔에서 환불 불가
- 환불은 고객 서비스(CS)를 통해서만 가능
- 문의: biz@adrop.io

- 정산 정보 미등록 시 수익금 지급 불가
