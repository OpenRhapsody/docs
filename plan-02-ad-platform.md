# 광고 플랫폼 배포

## ⚡ 퀵스타트: 광고 플랫폼 생성하기
- 🎬 데모 영상: 플랫폼 생성부터 공개까지
- ⏱️ 소요 시간: 약 3분
- 📋 시작 전 준비:
  - 서비스명 (2~20자)
  - 로고 이미지 (PNG/SVG, 256px 이상)
- ✅ 완료 후: 광고 플랫폼 링크 발급
- 👉 다음 단계: 광고 유닛 설정

## 플랫폼 생성하기
- 서비스 이름 입력
  - 공식 명칭 기입 (2~20자, 한글/영문/숫자)
  - 광고주 화면에 노출되므로 명확한 표기 필수
- 로고 및 테마 컬러 설정
  - PNG/SVG 형식 권장 (256px 이상)
  - 자동 추출된 테마 컬러는 플랫폼 전체에 적용
- 서브도메인 생성 (광고주 공유용 링크)
  - 영문 소문자, 숫자, 하이픈만 사용
  - 최종 링크: https://{입력값}.adrop.io
  - 중복 불가, 생성 후 수정 가능
- 운영 지역 설정 (국가, 시간대, 언어, 통화)
  - 예약/리포트/금액 표기에 반영
- 주의사항 (⚠️)
  - 계정 소유자/관리자만 생성 가능
  - 서비스별 각각 플랫폼 생성 가능
  - 로고 배경색 대비가 낮으면 테두리 보정 권장

## 플랫폼 설정하기
- 플랫폼 정보 수정

## 대시보드
- 캠페인 현황
  - 월별 신규 진행 캠페인 수
  - 종료 예정 캠페인 수
  - 수익금 정보
- 캠페인 성과
  - 노출 수
  - 클릭 수
  - 평균 클릭률
- Backfill 광고
  - Backfill 광고 수익
  - 노출 수 현황
- 데이터 갱신 주기: 1시간

## 결제 및 청구
- 연락처 관리
  - 기본 연락처 (Primary): 일반 안내 수신
  - 청구 연락처 (Invoice): 청구 관련 안내 수신
  - 입금 연락처 (Payout): 정산 관련 안내 수신
- 청구 정보
  - 국가 (광고 프로필 국가 설정 자동 반영)
  - 도시, 사업자명, TAX ID (사업자등록번호)
  - 사업자등록증 업로드
- 수익금 지급 계좌
  - 예금주명, 은행, 계좌번호
  - 통장 사본
  - SWIFT 코드 (해외 결제 시)

## 멤버 관리
- 멤버 초대
  - 이메일 입력 후 권한 선택하여 초대
  - 초대 안내 이메일 발송
- 멤버 역할
  - 소유자 (Owner): 모든 권한
  - 편집자 (Editor): 삭제/플랜변경/멤버관리 제외
  - 뷰어 (Viewer): 조회만 가능
  - 게스트 (Guest): 대시보드/캠페인 성과만 조회
