# 광고 플랫폼 배포

광고 플랫폼은 광고주가 접속하여 캠페인을 예약하는 공간입니다. 매체사는 자신만의 광고 플랫폼을 생성하고, 광고주에게 링크를 공유할 수 있습니다.

---

## ⚡ 퀵스타트: 광고 플랫폼 생성하기

> ⏱️ 소요 시간: 약 3분

### 시작 전 준비
- 서비스명 (2~20자)
- 로고 이미지 (PNG/SVG, 256px 이상)

### 단계별 가이드

**1단계: 콘솔 접속**

[애드컨트롤 콘솔](https://console.adrop.io)에 로그인하세요.

**2단계: 광고 플랫폼 생성**

1. 좌측 메뉴에서 **[광고 플랫폼]** 클릭
2. **[플랫폼 생성]** 버튼 클릭
3. 서비스 이름 입력 (2~20자, 한글/영문/숫자)
4. 로고 이미지 업로드
5. 서브도메인 입력 (예: `myservice` → `https://myservice.adrop.io`)
6. 운영 지역 설정 (국가, 시간대, 언어, 통화)
7. **[생성]** 버튼 클릭

**3단계: 완료 확인**

생성이 완료되면 광고 플랫폼 링크가 발급됩니다.
이 링크를 광고주에게 공유하면 광고주가 직접 캠페인을 예약할 수 있습니다.

### 다음 단계

## 플랫폼 생성하기

### 서비스 이름
- 공식 명칭을 기입하세요 (2~20자)
- 한글, 영문, 숫자 사용 가능
- 광고주 화면에 노출되므로 명확하게 표기하세요

### 로고 및 테마 컬러
- **형식**: PNG 또는 SVG 권장
- **크기**: 256px 이상
- **테마 컬러**: 로고에서 자동 추출되어 플랫폼 전체에 적용됩니다
- **팁**: 로고 배경색 대비가 낮으면 테두리 보정을 권장합니다

### 서브도메인
광고주에게 공유할 링크입니다.

- **사용 가능 문자**: 영문 소문자, 숫자, 하이픈(-)
- **최종 링크**: `https://{입력값}.adrop.io`
- **중복 불가**: 다른 매체사가 사용 중인 도메인은 사용할 수 없습니다
- **수정 가능**: 생성 후에도 변경할 수 있습니다

### 운영 지역 설정
| 항목 | 설명 | 영향 |
|------|------|------|
| 국가 | 서비스 운영 국가 | 세율 계산, 통화 기본값 |
| 시간대 | 캠페인 시작/종료 기준 | 예약 일정, 리포트 시간 |
| 언어 | 플랫폼 표시 언어 | 광고주 화면 언어 |
| 통화 | 금액 표시 단위 | 가격, 정산 금액 표기 |

### 주의사항
⚠️ 계정 소유자(Owner) 또는 편집자(Editor)만 플랫폼을 생성할 수 있습니다.

⚠️ 하나의 계정에서 여러 서비스를 운영하는 경우, 서비스별로 각각 플랫폼을 생성할 수 있습니다.

## 플랫폼 설정하기

생성된 플랫폼의 정보를 수정할 수 있습니다.

### 수정 가능 항목
- 서비스 이름
- 로고 이미지
- 테마 컬러
- 서브도메인
- 운영 지역 설정

### 수정 방법
1. 좌측 메뉴에서 **[광고 플랫폼]** 클릭
2. 수정할 플랫폼 선택
3. **[설정]** 탭 클릭
4. 정보 수정 후 **[저장]** 클릭

## 대시보드

플랫폼의 전체 현황을 한눈에 확인할 수 있습니다.

### 캠페인 현황
| 지표 | 설명 |
|------|------|
| 신규 진행 캠페인 | 이번 달 새로 시작된 캠페인 수 |
| 종료 예정 캠페인 | 이번 달 종료 예정인 캠페인 수 |
| 수익금 정보 | 이번 달 예상 수익금 |

### 캠페인 성과
| 지표 | 설명 |
|------|------|
| 노출 수 | 광고가 화면에 표시된 횟수 |
| 클릭 수 | 광고가 클릭된 횟수 |
| 평균 클릭률 (CTR) | 클릭 수 ÷ 노출 수 × 100 |

### Backfill 광고
| 지표 | 설명 |
|------|------|
| Backfill 수익 | 백필 광고에서 발생한 수익 |
| 노출 수 현황 | 백필 광고 노출 횟수 |

### 데이터 갱신 주기
대시보드 데이터는 **1시간** 단위로 갱신됩니다.

## 결제 및 청구

정산을 받기 위한 청구 정보와 계좌 정보를 등록합니다.

### 연락처 관리

| 연락처 유형 | 용도 |
|-------------|------|
| 기본 연락처 (Primary) | 일반 안내 수신 |
| 청구 연락처 (Invoice) | 청구 관련 안내 수신 |
| 입금 연락처 (Payout) | 정산 관련 안내 수신 |

### 청구 정보
정산서 및 세금계산서 발행에 필요한 정보입니다.

| 항목 | 설명 |
|------|------|
| 국가 | 광고 프로필 국가 설정이 자동 반영됩니다 |
| 도시 | 사업장 소재지 |
| 사업자명 | 세금계산서에 표기될 상호명 |
| TAX ID | 사업자등록번호 |
| 사업자등록증 | 파일 업로드 |

### 수익금 지급 계좌
정산금을 받을 계좌 정보입니다.

| 항목 | 설명 |
|------|------|
| 예금주명 | 계좌 소유자 이름 |
| 은행 | 은행 선택 |
| 계좌번호 | 정산금 입금 계좌 |
| 통장 사본 | 계좌 확인용 이미지 업로드 |
| SWIFT 코드 | 해외 송금 시 필요 (국내 계좌는 불필요) |

## 멤버 관리

팀원을 초대하고 권한을 관리합니다.

### 멤버 초대
1. **[관리]** → **[멤버]** 메뉴로 이동
2. 초대할 멤버의 이메일 입력
3. 부여할 권한 선택
4. **[초대]** 버튼 클릭
5. 초대 안내 이메일이 발송됩니다

### 멤버 역할

| 역할 | 설명 | 주요 권한 |
|------|------|----------|
| 소유자 (Owner) | 모든 권한 보유 | 멤버 관리, 플랜 변경, 삭제 |
| 편집자 (Editor) | 대부분의 편집 권한 | 캠페인 관리, 설정 변경 |
| 뷰어 (Viewer) | 조회만 가능 | 모든 데이터 조회 |
| 게스트 (Guest) | 제한된 조회 | 대시보드, 캠페인 성과만 조회 |

### 권한 상세

| 기능 | Owner | Editor | Viewer | Guest |
|------|:-----:|:------:|:------:|:-----:|
| 전체 데이터 조회 | ✔ | ✔ | ✔ | - |
| 대시보드/성과 조회 | ✔ | ✔ | ✔ | ✔ |
| 캠페인 관리 | ✔ | ✔ | - | - |
| 설정 변경 | ✔ | ✔ | - | - |
| 항목 삭제 | ✔ | - | - | - |
| 멤버 관리 | ✔ | - | - | - |
| 플랜/결제 변경 | ✔ | - | - | - |
