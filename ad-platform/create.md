# 광고 플랫폼 생성하기

## 생성 방법

<img src="./assets/ad-platform-launcher.gif" width="720" />

1. 좌측 상단의 프로젝트 이름 클릭
2. 드롭다운 메뉴 하단의 **[새로운 광고 플랫폼 배포하기]** 클릭

---

## 서비스 이름
- 공식 명칭을 기입하세요 (2~20자)
- 한글, 영문, 숫자 사용 가능
- 광고주 화면에 노출되므로 명확하게 표기하세요

## 로고 및 테마 컬러
- **형식**: PNG 또는 SVG 권장
- **크기**: 256px 이상
- **테마 컬러**: 로고에서 자동 추출되어 광고 플랫폼 전체에 적용됩니다
- **팁**: 로고 배경색 대비가 낮으면 테두리 보정을 권장합니다

## 서브도메인
광고주에게 공유할 링크입니다.

- **사용 가능 문자**: 영문 소문자, 숫자, 하이픈(-)
- **최종 링크**: `https://{입력값}.adrop.io`
- **중복 불가**: 다른 매체사가 사용 중인 도메인은 사용할 수 없습니다
- **수정 가능**: 생성 후에도 변경할 수 있습니다

## 운영 지역 설정
| 항목 | 설명 |
|------|------|
| 국가 | 회계 기준 지역을 선택하세요. 정산과 청구 정보에 사용되며, 통화는 국가 기본값을 따릅니다. |
| 시간대 | 콘솔에서 캠페인 예약이나 리포트를 볼 때 적용되는 시간대입니다. |
| 언어 | 콘솔에서 사용될 언어입니다. |

> 💡 광고주에게는 광고주의 설정에 맞는 시간대와 언어로 표기됩니다.

> ⚠️ 하나의 광고 플랫폼에는 단일 서비스만 등록하세요. 플랫폼별 앱(Android/iOS/Web)은 함께 운영할 수 있으나, 서로 다른 서비스가 혼합되면 타겟팅 데이터의 정확도에 영향을 줍니다.

## FAQ

**Q. 광고 플랫폼을 여러 개 만들 수 있나요?**

네. 서비스별로 별도의 광고 플랫폼을 생성할 수 있습니다. 각 플랫폼은 독립적인 브랜딩, 광고 상품, 정산 체계를 가집니다.

**Q. 커스텀 도메인을 사용할 수 있나요?**

기본적으로 `{서브도메인}.adrop.io` 형식으로 제공됩니다. 커스텀 도메인이 필요하시면 별도 문의해 주세요.

**Q. 광고주가 직접 캠페인을 예약할 수 있나요?**

네. 광고 플랫폼 링크를 공유하면 광고주가 직접 접속하여 상품 확인, 캠페인 예약, 소재 등록, 결제까지 셀프서비스로 진행할 수 있습니다.
