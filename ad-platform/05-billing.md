# 결제 및 청구

정산을 받기 위한 청구 정보와 수익금 지급 계좌를 등록합니다.

## 설정 방법

1. 좌측 메뉴에서 **[설정]** 클릭
2. **[결제 및 청구]** 탭 선택

---

## 청구 정보

정산서 및 세금계산서 발행에 필요한 정보입니다.

| 항목 | 설명 |
|------|------|
| 사업자명 | 세금계산서에 표기될 상호명 |
| TAX ID | 사업자등록번호 또는 세금 식별 번호 |
| 사업자등록증 | 파일 업로드 |
| 정산 대표 메일 | 세금계산서 발행 및 수신 안내를 받을 이메일 |
| 정산 참조 메일 | 참조로 받을 추가 이메일 (쉼표로 구분) |

> 💡 **국가별 TAX ID**
> - 한국: 사업자등록번호
> - 미국: EIN / TIN
> - EU: VAT Number
> - 일본: 法人番号
> - 인도: GSTIN / PAN

## 수익금 지급 계좌

정산금을 받을 계좌 정보입니다.

| 항목 | 설명 |
|------|------|
| 예금주명 | 계좌 소유자 이름 |
| 은행 | 은행 선택 |
| 계좌번호 | 정산금 입금 계좌 |
| 통장 사본 | 계좌 확인용 이미지 업로드 |
| SWIFT 코드 | 해외 송금 시 필요 (국내 계좌는 불필요) |

## FAQ

**Q. 광고 수익은 언제 정산되나요?**

카드 결제 캠페인은 매월 자동 정산됩니다. 캠페인 종료 후 정산서가 발행되며, 등록된 계좌로 수익금이 지급됩니다.

**Q. 수동 결제와 카드 결제의 차이는 무엇인가요?**

카드 결제는 애드컨트롤이 광고주로부터 결제를 받아 정산까지 자동 처리합니다. 수동 결제는 매체사가 광고주와 직접 정산을 진행합니다.

**Q. 해외 광고주도 결제할 수 있나요?**

네. 해외 결제는 Stripe를 통해 진행되며, 해외 결제 대행 수수료와 국가별 원천징수 세금이 적용될 수 있습니다.
