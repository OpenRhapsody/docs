# 대시보드

광고 플랫폼의 전체 현황을 한눈에 확인할 수 있습니다.

## 사용 방법

[▶ 영상 보기](./assets/ad-platform-dashboard.mp4)

1. 좌측 메뉴에서 **[대시보드]** 클릭
2. 상단에서 조회할 월 선택

---

## 캠페인 현황

해당 월의 캠페인 예약 및 수익 현황입니다.

| 지표 | 설명 |
|------|------|
| 신규 캠페인 | 해당 월에 새로 예약된 캠페인 수 |
| 총 예약금 | 신규 캠페인의 총 예약 금액 |
| 종료 캠페인 | 해당 월에 종료된 캠페인 수 |
| 예상 수익 / 수익 | 종료 캠페인의 정산 예상 금액 (당월은 '예상 수익', 과거는 '수익') |

> 💡 당월 조회 시 전월 대비 증감률이 함께 표시됩니다.

## 캠페인 성과

해당 월의 광고 노출 및 클릭 성과입니다.

| 지표 | 설명 |
|------|------|
| 총 노출 수 | 광고가 화면에 표시된 횟수 |
| 총 클릭 수 | 광고가 클릭된 횟수 |
| 평균 클릭률 | (클릭 수 ÷ 노출 수) × 100 |

## Backfill 광고

Backfill 광고의 수익 현황입니다.

| 지표 | 설명 |
|------|------|
| 예상 수익 | Backfill 광고에서 발생한 예상 수익 (USD 기준) |
| 총 노출 수 | Backfill 광고 노출 횟수 |

> 💡 Backfill 광고 노출은 애드컨트롤 사용량에 포함되지 않아 추가 과금이 발생하지 않습니다.

## 알림

대시보드 상단에 조치가 필요한 항목이 표시됩니다.

| 알림 | 설명 |
|------|------|
| 캠페인 심사가 필요해요 | 승인 대기 중인 캠페인이 있습니다. |
| 소재 심사가 필요해요 | 승인 대기 중인 소재가 있습니다. |
| 필수 소재 추가가 필요한 캠페인이 있어요 | 소재 미등록 캠페인이 있습니다. |
| 임박한 캠페인이에요 | 곧 시작되는 캠페인이 있습니다. |
| 결제 실패로 예약이 완료되지 않은 캠페인이 있어요 | 결제 재시도가 필요합니다. |

## FAQ

**Q. Backfill 광고란 무엇인가요?**

직광고가 송출되지 않는 인벤토리에 자동으로 네트워크 광고를 채워 Fill Rate를 100%에 가깝게 유지하는 기능입니다. 직광고 수익은 그대로 유지하면서 남는 트래픽까지 수익화할 수 있습니다.

**Q. 직광고와 애드몹(네트워크 광고) 수익 차이는 얼마나 되나요?**

일반적으로 직광고는 네트워크 광고 대비 2~10배 높은 CPM을 기록합니다. 애드컨트롤은 직광고를 우선 송출하고, 남는 인벤토리만 Backfill로 채워 전체 수익을 극대화합니다.

**Q. 광고 성과 데이터는 실시간으로 확인할 수 있나요?**

대시보드 데이터는 약 1시간 단위로 갱신됩니다. 실시간에 가까운 성과 확인이 필요하면 리포트 메뉴에서 더 상세한 데이터를 조회할 수 있습니다.
