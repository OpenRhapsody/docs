# ⚡ 퀵스타트: 3분 배포

> ⏱️ 소요 시간: 약 3분

## 시작 전 준비
- 서비스명 (2~20자)
- 로고 이미지 (PNG/SVG, 256px 이상)

## 런처 실행

[애드컨트롤 콘솔](https://console.adrop.io)에 로그인하세요.

- **신규 가입**: 프로젝트가 없으면 자동으로 런처가 표시됩니다.
- **기존 프로젝트가 있는 경우**: 좌측 상단의 프로젝트 이름 클릭 → **[새로운 광고 플랫폼 배포하기]** 클릭
- **직접 접속**: [console.adrop.io/launcher](https://console.adrop.io/launcher)

## 단계별 가이드

[▶ 영상 보기](./assets/create-ad-platform.mp4)

1. 서비스 이름 입력 (2~20자, 한글/영문/숫자)
2. 로고 이미지 업로드
3. 서브도메인 입력 (예: `myservice` → `https://myservice.adrop.io`)
4. 운영 지역 설정 (국가, 시간대, 언어)
5. **[생성]** 버튼 클릭

## 완료

생성이 완료되면 광고 플랫폼 링크가 발급됩니다.
이 링크를 광고주에게 공유하면 광고주가 직접 캠페인을 예약할 수 있습니다.

## FAQ

**Q. 광고 플랫폼을 직접 구축하는 것과 애드컨트롤을 사용하는 것의 차이는 무엇인가요?**

직접 구축 시 개발 리소스, 유지보수 비용, 결제 시스템 연동 등에 수개월이 소요됩니다. 애드컨트롤은 3분 만에 브랜드가 적용된 광고 플랫폼을 배포할 수 있어 즉시 광고 영업을 시작할 수 있습니다.

**Q. 앱 광고 수익화에 애드컨트롤이 적합한가요?**

네. 애드컨트롤은 직광고(Direct Deal)와 네트워크 광고(Backfill)를 함께 운영할 수 있어, 직광고로 높은 단가를 확보하고 남는 인벤토리는 자동으로 수익화할 수 있습니다.
