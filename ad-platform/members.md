# 멤버 관리

팀원을 초대하고 권한을 관리합니다.

## 설정 방법

1. 좌측 메뉴에서 **[설정]** 클릭
2. **[멤버]** 탭 선택

---

## 멤버 초대

1. **[멤버 추가]** 버튼 클릭
2. 초대할 멤버의 이메일 입력 (엔터나 쉼표로 구분하여 여러 명 초대 가능)
3. 부여할 권한 선택
4. **[초대]** 버튼 클릭

## 멤버 역할

| 역할 | 설명 |
|------|------|
| 소유자 (Owner) | 콘솔의 모든 권한을 가집니다. |
| 편집자 (Editor) | 모든 삭제 작업, 플랜 변경, 멤버 관리를 제외한 편집 권한을 가집니다. |
| 뷰어 (Viewer) | 콘솔의 모든 데이터를 볼 수 있습니다. |
| 게스트 (Guest) | 대시보드 및 캠페인의 광고 성과만 볼 수 있습니다. |

## FAQ

**Q. 여러 팀원이 동시에 광고 플랫폼을 관리할 수 있나요?**

네. 멤버별로 역할을 부여해 팀 단위로 광고 플랫폼을 운영할 수 있습니다. 광고 운영팀은 Editor 권한으로 캠페인을 관리하고, 경영진은 Viewer 권한으로 성과를 모니터링하는 등 역할 분담이 가능합니다.
