# 대시보드

플랫폼의 전체 현황을 한눈에 확인할 수 있습니다.

## 캠페인 현황
| 지표 | 설명 |
|------|------|
| 신규 진행 캠페인 | 이번 달 새로 시작된 캠페인 수 |
| 종료 예정 캠페인 | 이번 달 종료 예정인 캠페인 수 |
| 수익금 정보 | 이번 달 예상 수익금 |

## 캠페인 성과
| 지표 | 설명 |
|------|------|
| 노출 수 | 광고가 화면에 표시된 횟수 |
| 클릭 수 | 광고가 클릭된 횟수 |
| 평균 클릭률 (CTR) | 클릭 수 ÷ 노출 수 × 100 |

## Backfill 광고
| 지표 | 설명 |
|------|------|
| Backfill 수익 | 백필 광고에서 발생한 수익 |
| 노출 수 현황 | 백필 광고 노출 횟수 |

## 데이터 갱신 주기
대시보드 데이터는 **1시간** 단위로 갱신됩니다.
