# 멤버 관리

팀원을 초대하고 권한을 관리합니다.

## 멤버 초대
1. **[관리]** → **[멤버]** 메뉴로 이동
2. 초대할 멤버의 이메일 입력
3. 부여할 권한 선택
4. **[초대]** 버튼 클릭
5. 초대 안내 이메일이 발송됩니다

## 멤버 역할

| 역할 | 설명 | 주요 권한 |
|------|------|----------|
| 소유자 (Owner) | 모든 권한 보유 | 멤버 관리, 플랜 변경, 삭제 |
| 편집자 (Editor) | 대부분의 편집 권한 | 캠페인 관리, 설정 변경 |
| 뷰어 (Viewer) | 조회만 가능 | 모든 데이터 조회 |
| 게스트 (Guest) | 제한된 조회 | 대시보드, 캠페인 성과만 조회 |

## 권한 상세

| 기능 | Owner | Editor | Viewer | Guest |
|------|:-----:|:------:|:------:|:-----:|
| 전체 데이터 조회 | ✔ | ✔ | ✔ | - |
| 대시보드/성과 조회 | ✔ | ✔ | ✔ | ✔ |
| 캠페인 관리 | ✔ | ✔ | - | - |
| 설정 변경 | ✔ | ✔ | - | - |
| 항목 삭제 | ✔ | - | - | - |
| 멤버 관리 | ✔ | - | - | - |
| 플랜/결제 변경 | ✔ | - | - | - |
