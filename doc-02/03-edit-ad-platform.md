# 광고 플랫폼 설정하기

생성된 광고 플랫폼의 정보를 수정할 수 있습니다.

## 수정 방법
1. 좌측 메뉴에서 **[설정]** 클릭
2. **[광고 프로필]** 탭 선택
3. 정보 수정 후 **[저장]** 클릭

---

## 광고 플랫폼 정보

### 기본 정보

| 항목 | 설명 |
|------|------|
| 광고 플랫폼 이름 | 광고주에게 표시되는 광고 플랫폼 이름 |
| 대표 링크 | 광고주가 광고 플랫폼에 접속하는 URL (서브도메인) |

### 테마

| 항목 | 설명 |
|------|------|
| 앱 이미지 | 256x256px 이상, JPG/PNG/WebP 지원 |
| 브랜드 컬러 | 버튼과 배경에 적용되는 컬러. 너무 밝은 색상은 권장하지 않음 |
| 썸네일 | 링크 공유 시 표시되는 이미지. 600x315px 이상 |

### 문의 담당자 정보

| 항목 | 설명 |
|------|------|
| 대표 이메일 | 모든 문의와 안내 메일을 받을 이메일 |

### 지역 설정

| 항목 | 설명 |
|------|------|
| 국가 | 회계 기준 지역입니다. 정산과 청구 정보에 사용되며, 통화는 국가 기본값을 따릅니다. |
| 시간대 | 캠페인 예약, 리포트에 적용되는 시간대입니다. |
| 언어 | 콘솔에서 사용되는 언어입니다. |

> 💡 광고주에게는 광고주의 설정에 맞는 시간대와 언어로 표기됩니다.

## 서비스 정보

### 광고 상품 소개

광고 상품 소개서를 PDF로 업로드할 수 있습니다. 광고 플랫폼과 애드노트에서 광고 상품 안내에 활용됩니다.

애드노트용 소개서가 업로드되면 **A.drop AI Agent**가 내용을 분석하여 광고주 문의에 자동 응답하고 적합한 상품을 추천합니다.

#### 애드노트 등록 시 주의사항

| 항목 | 설명 |
|------|------|
| 연락처 | 직접 연락처 기재 금지. `contact@adrop.io`로만 표기 |
| 상품명 | 소개서의 상품명이 콘솔 상품명과 일치하도록 수정 |
| 비공개 정보 | 외부 공유가 불가능한 정보는 모두 제외 |
| 정책 | 공식 정책과 다른 개별 매체 정책, 매체사만의 예약 프로세스 등은 제외 |

### 서비스 소개

광고 플랫폼과 애드노트에서 서비스 안내에 활용됩니다.

| 항목 | 설명 |
|------|------|
| 한 줄 소개 | 서비스만의 차별화된 도달력과 핵심 타겟 강점을 표현합니다. |
| 자세한 소개 | 어떤 캠페인에 서비스가 특히 잘 맞는지 설명합니다. |
