# 결제 및 청구

정산을 받기 위한 청구 정보와 계좌 정보를 등록합니다.

## 연락처 관리

| 연락처 유형 | 용도 |
|-------------|------|
| 기본 연락처 (Primary) | 일반 안내 수신 |
| 청구 연락처 (Invoice) | 청구 관련 안내 수신 |
| 입금 연락처 (Payout) | 정산 관련 안내 수신 |

## 청구 정보
정산서 및 세금계산서 발행에 필요한 정보입니다.

| 항목 | 설명 |
|------|------|
| 국가 | 광고 프로필 국가 설정이 자동 반영됩니다 |
| 도시 | 사업장 소재지 |
| 사업자명 | 세금계산서에 표기될 상호명 |
| TAX ID | 사업자등록번호 |
| 사업자등록증 | 파일 업로드 |

## 수익금 지급 계좌
정산금을 받을 계좌 정보입니다.

| 항목 | 설명 |
|------|------|
| 예금주명 | 계좌 소유자 이름 |
| 은행 | 은행 선택 |
| 계좌번호 | 정산금 입금 계좌 |
| 통장 사본 | 계좌 확인용 이미지 업로드 |
| SWIFT 코드 | 해외 송금 시 필요 (국내 계좌는 불필요) |
