# 분석 및 리포트

## ⚡ 퀵스타트: 리포트 확인하기
- 🎬 데모 영상: 리포트 조회부터 다운로드까지
- ⏱️ 소요 시간: 약 3분
- 📋 시작 전 준비:
  - 진행 중이거나 종료된 캠페인
- ✅ 완료 후: 성과 데이터 확인 및 다운로드
- 👉 다음 단계: 정산 확인

## 리포트란?
- 캠페인/상품/유닛/소재별 성과 데이터 조회
- 광고주와 공유 가능한 데이터 제공
- 데이터 갱신 주기: 10분

## 지표 이해하기

### 기본 지표
- 노출수 (Impressions)
  - 광고가 화면에 표시된 횟수
  - 디스플레이/비디오 구분 가능
- 클릭수 (Clicks)
  - 광고가 클릭된 횟수
  - 디스플레이/비디오 구분 가능
- 해제 수 (Dismisses)
  - 광고가 해제된 횟수
- CTR (클릭률)
  - 클릭수 ÷ 노출수 × 100
  - 디스플레이/비디오 별도 계산 가능

### 비용 지표
- CPM (Cost Per Mille)
  - 1,000회 노출당 비용
- CPC (Cost Per Click)
  - 클릭 1회당 비용
- CPP (Cost Per Period)
  - 기간당 비용

### 정산 지표 (매체사)
- 예산 (Supply Amount)
  - 캠페인 설정 예산
- 환불액 (Refund Amount)
  - 목표 대비 미달성분 환불
- 수수료 (Commission)
  - 대행 수수료 (해당 시)
- 정산액 (Settlement)
  - 최종 정산 금액

## 리포트 조회하기

### 조회 기준 (탭)
- 캠페인별
  - 유닛/소재 세그먼트 성과
- 상품별
  - 캠페인/유닛/소재 세그먼트 성과
- 유닛별
  - 캠페인/상품/소재 세그먼트 성과
- 소재별
  - 유닛별 세그먼트 성과
- Backfill
  - 유닛별 Backfill 광고 수익
  - 노출수, 클릭수, CTR, CPC, CPM, 수익금

### 세그먼트
- 탭 내에서 추가 분류 가능
- 예: 캠페인별 → 유닛별 세그먼트

### 필터링
- 날짜/시간대 필터
- 캠페인/상품/유닛/소재 필터
- 복수 선택 가능

### 롤업 (집계 단위)
- 날짜별 소계
- 시간별 소계
- 총계만

### 정렬
- 노출수 기준
- 클릭수 기준
- 오름차순/내림차순

### 열 편집
- 표시할 컬럼 선택

## 리포트 다운로드

### CSV 내보내기
- 리포트 데이터 다운로드
- 파일명 형식: report-{탭}_{세그먼트}-{시작일}_{종료일}.csv
- 이메일로 다운로드 링크 발송

### 컬럼 선택
- 리포트 다운로드 시 필요한 필드만 선택 가능
- Tab/Segment별 기본 컬럼 제공
- 선택한 컬럼만 CSV에 포함

### 광고 유닛 목록 내보내기
- 프로젝트의 모든 광고 유닛 정보
- 파일명 형식: ad_units-{날짜}.csv

### 다운로드 대기
- 대용량 데이터는 백그라운드 처리
- 완료 시 이메일 알림

## 광고주 리포트 설정 (매체사)
- 프로젝트 단위로 광고주에게 제공할 리포트 범위 설정
- 설정 가능 항목
  - 캠페인 정보 포함 여부
  - 소재 정보 포함 여부
  - 광고 유닛 정보 포함 여부
  - 광고 상품 정보 포함 여부
  - 날짜 정보 포함 여부
  - 시간 정보 포함 여부
- 광고주/매체사 리포트 차이
  - 공통: 노출/클릭 지표, 캠페인 정보, 상품/유닛/소재 정보
  - 광고주만: 광고주명, 대행사 정보
  - 매체사만: 정산 정보, Backfill 광고 수익

## 리포트 컬럼 설명
- 기본 지표 컬럼
- 캠페인 정보 컬럼
- 정산 정보 컬럼
- 상품/유닛/소재 컬럼
