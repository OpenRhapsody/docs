# 캠페인 예약하기

## ⚡ 퀵스타트: 첫 캠페인 예약하기
- 🎬 데모 영상: 광고 계정 생성부터 캠페인 제출까지
- ⏱️ 소요 시간: 약 10분
- 📋 시작 전 준비:
  - 광고 플랫폼 링크 (매체사에서 제공)
  - 소재 파일 (이미지/동영상)
  - 결제 정보 (카드 또는 사업자 정보)
- ✅ 완료 후: 캠페인 심사 대기
- 👉 다음 단계: 심사 결과 확인

## 캠페인이란?
- 개념: 광고주가 집행하는 광고 단위
- 캠페인 상태 흐름
  - 작성중 → 심사중 → 거절됨/예약됨 → 진행중 → 정지됨/취소됨/종료됨

## 광고 계정 만들기

### 광고 플랫폼 접속
- 매체사가 제공한 링크로 접속
  - 형식: https://{매체사}.adrop.io
- 로그인 또는 회원가입

### 광고 계정 생성
- 광고 계정이란?
  - 캠페인을 예약하고 관리하는 단위
  - 하나의 조직에서 여러 광고 계정 생성 가능
- 계정 정보 입력
  - 계정 이름
  - 담당자 정보

### 광고주 프로필
- 광고주명
- 웹사이트 링크
- 대표 연락 이메일
- 국가, 시간대, 언어 설정
  - 세율, 캠페인 일정, 리포트 데이터 표시에 영향

### 결제 프로필 설정
- 결제 프로필이란?
  - 청구 정보와 결제 수단을 저장
  - 여러 광고 계정에서 재사용 가능
- 청구 정보 입력
  - 국가, 도시, 사업자명
  - TAX ID (사업자등록번호)
  - 사업자등록증 업로드
  - 대표 이메일
- 결제 수단 등록
  - 한국 카드: 토스페이먼츠 (원화, 3.5% 수수료)
  - 글로벌 카드: Stripe (다통화, 5.5% 수수료)
- 프로필 재사용
  - 새 광고 계정 생성 시 기존 프로필 선택 가능
  - 국가가 일치하는 프로필만 사용 가능

### 멤버 관리
- 이메일로 팀원 초대
- 역할 지정
- 역할 권한
  - Owner: 전체 플랫폼 접근, 결제 수단 관리, 계정 삭제, 멤버 권한 관리
  - Editor: 삭제 및 멤버 관리 제외한 모든 편집
  - Viewer: 데이터 조회만 가능
  - Guest: 대시보드 및 캠페인 성과 조회만 가능

## 캠페인 예약하기

### 광고 상품 선택
- 광고 상품 목록 확인
  - 매체사가 판매 중인 상품 조회
  - 상품별 가격, 예상 성과 확인
- 상품 상세 정보
  - 과금 유형 (CPM/CPC/CPP)
  - 노출 위치 및 형식
  - 소재 가이드

### 기간 설정
- 시작일/종료일 선택
  - 시간대 기준 확인 (매체사 설정)
  - 최소 7영업일 전 예약 권장 (원활한 심사/승인 위해)
- CPP 상품의 경우
  - 최소 구매 기간 확인
  - 슬롯 선택

### 예산 설정
- CPM/CPC 상품
  - 총 예산 금액 입력
  - 일 할당량 설정 (선택)
    - 자동: 기간 내 균등 분배 (매일 자정 자동 조정)
    - 수동: 직접 입력
- CPP 상품
  - 슬롯 수에 따른 고정 금액

### 타겟팅 선택
- 타겟팅 옵션 확인
  - 매체사가 제공하는 타겟팅 목록
- 오디언스 타겟팅
  - 사용자 속성 기반 (연령, 성별, 관심사 등)
- 문맥 타겟팅
  - 콘텐츠 카테고리 기반
- 할증률 확인
  - 타겟팅 선택 시 추가 비용 발생 가능
  - 복수 선택 시 할증률 누적

### 노출 빈도 제한 (선택)
- 동일 사용자에게 노출되는 횟수 제어
- 높게 설정 시 도달 범위 증가, 클릭률 하락 가능

## 소재 등록하기

### 소재 형식
- 이미지
  - 권장: 등록 사이즈의 3배 해상도
  - 지원 형식: JPG, PNG, GIF
- 동영상
  - 비율: 9:16 (세로), 16:9 (가로)
  - 길이: 5~30초
  - 파일 크기: 최대 1GB
  - 지원 형식: MP4, MOV, WebM, MKV, AVI
  - 해상도: 최소 1080p
  - 썸네일 이미지 필수 (JPG, PNG)
  - 스킵 지연 시간 확인
  - iOS: 하단 34dp Safe Area에 텍스트 배치 주의

### 소재 정보 입력
- 소재 이름
  - 성과 추적용 식별자
- 도착 URL
  - 브라우저: https://
  - 앱: appscheme://
- 도착 위치 (브라우저 설정)
  - 기본 브라우저: 사용자 기본 브라우저에서 열기
  - 인앱 브라우저: 앱 내 브라우저에서 열기
- 다크 모드 소재 (선택)
  - 다크 모드용 별도 이미지 등록

### 유닛 세트별 소재
- 유닛 세트란?
  - 동일 소재가 노출되는 지면 묶음
- 세트별 소재 등록
  - 각 유닛 세트에 맞는 소재 업로드

### 노출 스케줄 (선택)
- 노출 기간 지정
  - 캠페인 기간 내 특정 기간만 노출
- 노출 시간 지정
  - 요일/시간별 노출 설정
  - 매체사가 허용한 경우에만 사용 가능

## 결제 및 제출

### 예약 내용 확인
- 캠페인 정보 요약
  - 상품, 기간, 예산, 타겟팅
- 예상 비용 확인
  - 기본 비용 + 타겟팅 할증

### 결제 정보 선택
- 등록된 결제 프로필 선택
- 결제 방식
  - 카드 결제: 선결제, 자동 정산
  - 수동 결제: 매체사와 직접 정산 (매체사가 허용한 경우)

### 약관 동의 및 제출
- 캠페인 예약 약관 확인
- 제출 후 상태 변경
  - 작성중 → 심사중
- 심사 대기
  - 매체사가 캠페인 및 소재 심사
  - 승인 시 예약됨 상태로 변경

## 예약 후 확인하기

### 캠페인 상태 확인
- 심사중: 매체사 검토 대기
- 거절됨: 거절 사유 확인 후 수정 재제출
- 예약됨: 시작일까지 대기
- 진행중: 광고 노출 중

### 소재 상태 확인
- 심사중: 소재 검토 대기
- 승인됨: 노출 가능
- 거절됨: 거절 사유 확인 후 수정 재제출
- 거절 사유 종류
  - 법률 위반, 허위/과장 광고, 비방/명예훼손
  - 음란물, 폭력성, 저작권/초상권 침해
  - 사회 규범 위반, 청소년 유해, 정책 위반
  - 소재 오류, 기타

### 캠페인 수정
- 수정 가능 항목
  - 기간 (진행중에는 시작일 수정 불가)
  - 타겟팅
  - 캠페인 이름
  - 노출 빈도 제한
  - 일 할당량
- 수정 불가 항목
  - 광고 상품
  - 예산
  - 결제 정보

### 캠페인 일시정지/취소
- 매체사 승인 필요
- 담당자와 이메일로 협의

### 캠페인 복제
- 기존 캠페인 기반으로 새 캠페인 생성
- 복제 시 시작일은 현재로부터 7일 후로 설정됨
- 복제 후 수정하여 제출

## 알림
- 캠페인 시작 알림
  - 캠페인 시작 1시간 전 알림 발송
- 캠페인 종료 알림
  - 캠페인 종료 1시간 전 알림 발송
- 소재 액션 필요 알림
  - 캠페인 시작 7일 전부터 알림
  - 소재 미등록 또는 거절된 경우 알림 발송

## 스플래시 광고 예약 가이드

### 스플래시 광고란?
- 노출 중심 광고 (클릭 기능 없음)
- 랜딩 URL 불필요
- 리포트에서 클릭수는 항상 0
- 브랜드 런칭, 시즌 캠페인에 적합

### 72시간 사전 확정 규칙
- 캠페인 시작 72시간 전까지 소재·일정·옵션 확정 필요
- 확정된 정보가 사용자 기기에 미리 저장됨
- 시작 시 저장된 정보로 즉시 노출

### 72시간 이내 변경 시 (⚠️)
- 소재 변경: 일부 사용자에게 기존 소재 계속 노출
- 일정 변경/취소: 일부 사용자에게 기존 일정대로 노출
- 새 정보는 앱 재시작 후에만 반영
- 기존/신규 소재 노출 모두 과금 대상
- 위약금은 매체사 정책에 따름

## 동영상 광고 예약 가이드

### 동영상 소재 규격
- 비율: 9:16 (세로), 16:9 (가로)
- 길이: 5~30초
- 파일 크기: 최대 1GB
- 지원 형식: MP4, MOV, WebM, MKV, AVI
- 해상도: 최소 1080p

### 썸네일 이미지
- 형식: JPG, JPEG, PNG
- 크기: 세로 540×960px, 가로 960×540px 이상
- 비율: 영상과 동일
- 파일 크기: 최대 10MB

### 가격 구조
- 기본 CPM + 동영상 추가 요금
- 이미지/동영상 혼합 가능 (단가 상이)

### iOS 주의사항
- 하단 34dp Safe Area에 텍스트 배치 주의
- 인터페이스 요소와 겹칠 수 있음

### 규격 미준수 시 (⚠️)
- 소재 등록 불가
- 캠페인 진행 불가

## 주의사항
- 심사 기한
  - 캠페인 제출 후 72시간 이내 또는 시작 전까지 심사 완료
  - 기한 초과 시 자동 거절
- 당일 예약
  - 당일 시작 캠페인은 시작 후 24시간 내 미승인 시 자동 거절
  - 매체사가 당일 예약을 허용한 경우에만 가능
- 소재 미승인
  - 캠페인이 승인되어도 승인된 소재가 없으면 광고 송출 안 됨
- 복제 제약
  - 광고 상품의 유닛이 삭제된 경우 복제 불가
  - 유닛 세트에 수정이 필요한 경우 복제 불가

## 광고 정책

### 기본 원칙
- 현행 법률 및 정부 기관 권고 위반 광고 금지
- 승인된 광고도 정책 위반 시 사전 통보 없이 중단 가능
- 법적 책임은 광고주에게 있음

### 금지 콘텐츠

#### 허위/과장 광고
- 사실과 다르거나 지나치게 부풀린 내용
- 중요 조건 누락 또는 작은 글씨로 표시
- 객관적 근거 없는 최상급 표현

#### 비교 광고
- 비교 시 명확한 근거와 객관적 자료 필요
- 경쟁사 제품에 대한 근거 없는 주장 금지

#### 음란/선정적 콘텐츠
- 과도한 노출 또는 성적 암시
- 성폭력 묘사 또는 착취적 콘텐츠
- 성인 서비스 홍보

#### 폭력/혐오/공포
- 잔인한 폭력, 고어, 불쾌한 이미지
- 보호 대상 특성에 기반한 차별
- 인종, 종교, 성별 기반 혐오 메시지

#### 지식재산권 침해
- 특허, 저작권, 상표권, 초상권 침해 금지

#### 청소년 보호
- 청소년 유해물, 유해 약물, 도박 유도 금지
- 연령 부적합 성적 콘텐츠 금지
- 주류/담배 광고는 연령 제한 조치 필요

#### 가격 표시
- 원가는 정확해야 함
- 허위 할인율 금지
- 모든 사용자에게 동일한 가격 적용
