# 다크 모드 지원하기

앱의 다크 모드에 맞는 광고 소재를 제공하는 방법을 안내합니다.

<img src="./assets/darkmode.avif" alt="다크 모드 설정">

---

## 다크 모드란?

다크 모드는 어두운 배경에 밝은 텍스트를 사용하는 UI 테마입니다. 앱이 다크 모드일 때 광고도 어두운 톤의 소재로 노출하면 사용자 경험이 향상됩니다.

다크 모드 소재 받기 기능을 활성화하면 하나의 광고 유닛으로 라이트 모드와 다크 모드 환경 모두에 최적화된 광고를 운영할 수 있습니다.

---

## 지원 포맷

다음 광고 유닛에서 다크 모드 소재를 지원합니다.

- 배너
- 팝업
- 스플래시
- 네이티브

> ⚠️ 동영상 소재는 다크 모드를 지원하지 않습니다.

---

## 설정 방법

광고 유닛을 생성하거나 수정할 때 **다크 모드 소재 받기** 토글을 활성화합니다.

| 설정 | 동작 |
|------|------|
| ON | 다크 모드 소재 수신 가능 |
| OFF | 항상 기본(라이트) 소재만 수신 |

---

## 동작 방식

1. **광고 유닛 설정**: 광고 유닛에서 다크 모드 소재 받기를 활성화합니다.
2. **소재 등록**: 광고주가 캠페인 예약 시 라이트 모드 소재와 다크 모드 소재를 각각 등록합니다.
3. **SDK 요청**: 앱의 현재 테마(light/dark)를 SDK가 자동으로 감지하여 요청합니다.
4. **소재 응답**: 다크 모드 요청 시 다크 소재를 반환합니다.

### 폴백 동작

다크 모드 소재가 등록되지 않은 경우, 라이트 모드 소재가 노출됩니다.

---

## 주의사항

**캠페인 진행 중 설정 변경**

진행 중인 캠페인에 포함된 광고 유닛에서 다크 모드를 활성화하면, 라이트 모드 소재가 기본값으로 노출됩니다.

**소재 심사**

라이트 모드와 다크 모드 소재가 동시에 심사됩니다. 둘 중 하나라도 정책을 위반하면 소재 전체가 거절됩니다.
