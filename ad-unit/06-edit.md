# 광고 유닛 수정하기

생성된 광고 유닛의 설정을 변경하는 방법을 안내합니다.

---

## 수정 절차

1. **광고 유닛** 메뉴에서 수정할 유닛을 선택합니다.
2. 설정을 변경합니다.
3. **저장**을 클릭합니다.

---

## 수정 가능한 항목

| 항목 | 수정 가능 여부 |
|------|---------------|
| 유닛 이름 | O |
| 포맷별 옵션 | O |
| 다크 모드 지원 | O |
| 포맷 | X |
| 사이즈 | X |
| 광고 유닛 ID | X |

---

## 주의사항

- 포맷과 사이즈는 변경할 수 없습니다. 변경이 필요하면 새 광고 유닛을 생성하세요.
- 진행 중인 캠페인에 포함된 광고 유닛을 수정하면 즉시 반영됩니다.
- 다크 모드를 활성화하면 기존 캠페인은 라이트 모드 소재로 노출됩니다.
