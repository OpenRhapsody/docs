# 광고 유닛 관리하기

생성된 광고 유닛을 수정하거나 삭제하는 방법을 안내합니다.

> ⚠️ **수정 전 확인하세요**
>
> 광고 유닛 수정은 진행 중인 캠페인과 광고 상품에 영향을 줄 수 있습니다. 특히 팝업 광고는 수정 즉시 반영되므로 주의가 필요합니다. 가능하면 캠페인이 없는 시점에 수정하거나, 새 유닛을 생성하는 것을 권장합니다.

---

## 수정하기

### 수정 절차

1. **광고 유닛** 메뉴에서 수정할 유닛을 선택합니다.
2. 상세 페이지 우측 상단의 **[...]** 버튼을 클릭합니다.
3. **유닛 수정**을 선택합니다.
4. 설정을 변경하고 **저장**을 클릭합니다.

### 수정 가능한 항목

| 항목 | 수정 가능 여부 | 비고 |
|------|---------------|------|
| 유닛 이름 | O | |
| 포맷별 옵션 | O | |
| 다크 모드 지원 | O | |
| 포맷 | 조건부 | SDK 연동 전이고 광고 상품에 미포함 시 |
| 사이즈 | 조건부 | SDK 연동 전이고 광고 상품에 미포함 시 |
| 광고 유닛 ID | X | |

### 캠페인 반영 시점

> ⚠️ **포맷에 따라 반영 시점이 다릅니다.**

| 포맷 | 반영 시점 |
|------|----------|
| 팝업 | 운영 중인 캠페인에 **즉시 반영** |
| 배너, 전면, 보상형, 네이티브, 스플래시 | **새 캠페인부터 반영** |

팝업 광고는 수정 즉시 운영 중인 캠페인에 적용되므로 주의가 필요합니다.

### 수정 시 주의사항

- **포맷·사이즈 수정 제한**: SDK 연동 후이거나 광고 상품에 포함된 유닛은 포맷과 사이즈를 변경할 수 없습니다.
- **다크 모드 활성화**: 진행 중인 캠페인은 라이트 모드 소재로 노출됩니다.

---

## 삭제하기

### 삭제 절차

1. **광고 유닛** 메뉴에서 삭제할 유닛을 선택합니다.
2. 상세 페이지 우측 상단의 **[...]** 버튼을 클릭합니다.
3. **유닛 삭제**를 선택합니다.
4. 유닛 이름을 입력하여 삭제를 확인합니다.

### 삭제 시 주의사항

- 삭제된 광고 유닛은 **복구할 수 없습니다**.
- 진행 중인 캠페인에 포함된 광고 유닛은 삭제할 수 없습니다. 캠페인을 먼저 종료하세요.
- 삭제해도 해당 유닛의 과거 리포트 데이터는 유지됩니다.
- 삭제된 광고 유닛 ID로 광고를 요청하면 광고가 노출되지 않습니다.

## FAQ

**포맷이나 사이즈를 변경하고 싶은데 수정이 안 돼요**

SDK 연동 후이거나 광고 상품에 포함된 유닛은 포맷과 사이즈를 변경할 수 없습니다. 새 유닛을 생성하고 점진적으로 전환하는 것을 권장합니다.

**유닛을 수정했는데 운영 중인 캠페인에 반영이 안 돼요**

팝업을 제외한 포맷은 운영 중인 캠페인에 즉시 반영되지 않습니다. 수정 내용은 새로 생성하는 캠페인부터 적용됩니다.

**유닛을 수정했더니 캠페인 예약할 때 해당 상품이 안 보여요**

유닛의 포맷이나 사이즈가 광고 상품에 설정된 조건과 맞지 않으면 해당 상품이 캠페인 예약 목록에 나타나지 않습니다. 광고 상품 설정에서 유닛 세트 조건을 확인해주세요.

**유닛을 삭제하면 기존 캠페인에 영향이 있나요?**

이미 생성된 캠페인에는 영향이 없습니다. 삭제 이후에는 해당 유닛이 포함된 상품으로 새 캠페인을 예약할 수 없습니다.
