# 백필 광고로 수익 극대화하기

광고 인벤토리를 100% 활용하여 수익을 극대화하는 방법을 안내합니다.

---

## 왜 백필 광고가 필요한가요?

### 직광고만으로는 한계가 있습니다

직접 영업한 광고(Direct Deal)는 단가가 높지만, 모든 인벤토리를 채우기 어렵습니다. 광고가 없는 시간대에는 수익이 0원입니다.

### 네트워크 광고만으로도 부족합니다

AdMob, AppLovin 등 네트워크 광고는 빈 인벤토리를 채워주지만, 아무리 최적화해도 단가에는 한계가 있습니다. 네트워크 광고끼리의 경쟁만으로는 eCPM을 획기적으로 높이기 어렵습니다.

### 진짜 수익화는 둘의 조합입니다

| 광고 유형 | 장점 | 단점 |
|-----------|------|------|
| 직광고 (Direct) | 높은 단가 | Fill Rate 한계 |
| 네트워크 광고 | 높은 Fill Rate | 단가 한계 |
| **직광고 + 백필** | **높은 단가 + 높은 Fill Rate** | - |

**애드컨트롤의 백필 광고**는 직광고가 없는 인벤토리를 자동으로 네트워크 광고로 채워줍니다. 직광고의 높은 수익성과 네트워크 광고의 안정성을 동시에 확보할 수 있습니다.

---

## 백필 광고란?

직광고가 없어 비어있는 광고 인벤토리를 **네트워크 광고로 자동으로 채워주는 기능**입니다.

애드컨트롤의 비딩·워터폴 최적화 기술을 통해 광고 단가(eCPM)와 Fill Rate를 동시에 높여 안정적인 매출 상승을 제공합니다.

### 작동 방식

```
광고 요청 → 직광고 확인 → 직광고 있음 → 직광고 노출 (높은 단가)
                       → 직광고 없음 → 백필 광고 노출 (안정적 수익)
```

직광고가 우선 노출되고, 직광고가 없을 때만 백필 광고가 노출됩니다. 직광고 수익을 보호하면서 빈 인벤토리까지 수익화할 수 있습니다.

---

## 활성화하기

![백필 광고 활성화](./assets/backfill_ko.avif)

### 활성화 절차

1. **광고 유닛** 메뉴에서 대상 유닛을 선택합니다.
2. 상세 페이지에서 **백필 광고 켜기** 버튼을 클릭합니다.
3. 웹 SDK 연동 여부를 선택합니다.
4. 요청 후 상태가 **처리중**으로 변경됩니다.
5. 연결 완료 시 자동으로 수익화가 시작됩니다.

> 💡 **첫 활성화 시 안내**
>
> 최초 활성화 요청 시, 직광고와 백필 광고의 최적화된 운영을 위해 담당자가 연락드립니다.

### 일시정지 및 재시작

백필 광고는 언제든지 일시정지하고 다시 시작할 수 있습니다.

- 상태 변경 후 **최대 30분** 내에 반영됩니다.
- 직광고 캠페인에는 영향을 주지 않습니다.

## 지원 포맷

| 포맷 | 백필 지원 | 비고 |
|------|-----------|------|
| 배너 | O | |
| 전면 | O | |
| 보상형 | O | |
| 네이티브 | O | headline 또는 이미지 필수 |
| 팝업 | △ | 지원 예정 |
| 스플래시 | X | 직광고 전용 |

### 네이티브 유닛 주의사항

- 추가 텍스트 항목(부제목 등)은 백필 광고에서 표시되지 않습니다.
- **headline** 또는 **이미지**를 반드시 등록해야 백필 광고가 노출됩니다.

## 성과 확인

백필 광고의 성과는 **리포트 > 백필** 탭에서 확인할 수 있습니다.

| 지표 | 설명 |
|------|------|
| 노출수 | 백필 광고가 노출된 횟수 |
| Fill Rate | 광고 요청 대비 노출 비율 |
| 클릭수 | 백필 광고 클릭 횟수 |
| CTR | 클릭률 (클릭수 / 노출수) |
| CPC | 클릭당 수익 |
| eCPM | 1,000회 노출당 수익 |
| 수익 | 백필 광고로 발생한 수익 |

## 수익화 전략

### 1단계: 백필 광고로 기반 수익 확보

백필 광고를 활성화하여 모든 인벤토리에서 기본 수익을 확보하세요. 광고가 비어있는 시간이 사라집니다.

### 2단계: 직광고로 수익 상승

광고 상품을 등록하고 광고주를 유치하세요. 직광고가 들어오면 백필보다 높은 단가로 수익이 상승합니다.

### 3단계: 포트폴리오 최적화

리포트에서 직광고와 백필 광고의 비율을 분석하고, 수익성이 높은 유닛에 직광고 영업을 집중하세요.

> 💡 **목표**
>
> 직광고 비중을 높여가면서 전체 eCPM을 지속적으로 상승시키는 것이 핵심입니다. 애드컨트롤은 이 과정을 하나의 플랫폼에서 관리할 수 있게 해줍니다.

## FAQ

**직광고가 들어오면 기존 백필 광고는 어떻게 되나요?**

자동으로 직광고가 우선 노출되고, 직광고가 없는 인벤토리에만 백필이 채워집니다. 수동 전환 작업이 필요 없습니다. 다른 솔루션에서는 직광고가 들어올 때마다 개발자가 코드를 수정해야 하지만, 애드컨트롤은 하나의 광고 유닛에서 직광고와 백필이 자동으로 조율됩니다.

**다른 네트워크 광고와 함께 사용해도 되나요?**

권장하지 않습니다. 애드컨트롤 광고 유닛으로 통일해야 직광고와 네트워크 광고를 실시간으로 전환하며 수익을 극대화할 수 있습니다. 여러 솔루션을 혼용하면 운영과 개발에 불필요한 리소스가 들고, 통합 리포트와 최적화의 이점을 누릴 수 없습니다. 광고 사업을 본격적으로 확장하려면 하나의 플랫폼에서 직광고와 백필을 함께 관리하는 것이 효율적입니다.

**직광고 영업이 어려운데 백필만 먼저 써도 되나요?**

오히려 권장합니다. 백필로 먼저 수익화를 시작하면서 트래픽 데이터를 축적하세요. 이 데이터가 직광고 영업 시 강력한 협상 자료가 됩니다. 광고주가 확보되면 콘솔에서 광고 상품만 등록하면 됩니다. 별도 개발 없이 바로 직광고 수익화로 전환할 수 있습니다.

**직광고 단가가 백필보다 훨씬 높은 이유가 뭔가요?**

네트워크 광고는 여러 앱에 동시 노출되는 광고를 경매 방식으로 받기 때문에 단가에 한계가 있습니다. 반면 직광고는 광고주가 특정 앱·지면을 지정해서 구매하므로 프리미엄 단가를 받을 수 있습니다. 백필로 기반 수익을 확보하면서 직광고 비중을 높여가는 것이 수익 극대화의 핵심입니다.
