# 소재 등록하기

캠페인에 광고 소재를 등록하는 방법을 설명합니다. **소재**는 사용자에게 노출되는 광고 콘텐츠로, 이미지, 동영상, 텍스트 등으로 구성됩니다.

[▶ 영상 보기](./assets/campaign-creative.mp4)

---

## 소재 섹션

캠페인 상세 화면 하단에 소재 섹션이 있습니다.

### 필수 소재

광고 상품에 따라 등록해야 할 소재 유형이 표시됩니다. 각 카드에는 소재 유형, 규격, 등록 버튼이 표시됩니다.

### 등록된 소재 목록

이미 등록된 소재가 테이블로 표시됩니다. 소재를 클릭하면 상세 정보를 확인하고 수정할 수 있습니다.

---

## 소재 등록하기

필수 소재 카드의 **소재 등록** 버튼을 클릭하여 소재 등록 화면을 엽니다.

### 공통 입력 항목

모든 광고 포맷에서 입력하는 항목입니다.

#### 파일 업로드

파일을 드래그 앤 드롭하거나 **파일 선택**을 클릭하여 업로드합니다. 업로드 후 규격 검증 결과를 확인하고, 오류가 있으면 수정 후 다시 업로드합니다.

**이미지 소재**

| 항목 | 규격 |
|------|------|
| **포맷** | JPG, PNG, WebP, GIF |
| **최대 용량** | 10 MB |
| **최소 해상도** | 광고 규격의 3배 이상 |
| **비율** | 광고 규격과 동일 (1% 오차 허용) |

권장 해상도 예시:

| 광고 규격 (dp) | 최소 해상도 (px) |
|--------------|----------------|
| 300×250 | 900×750 이상 |
| 320×180 (16:9) | 960×540 이상 |

> ⚠️ 스플래시 광고는 GIF를 지원하지 않습니다.

**다크 모드 소재**

매체사가 다크 모드를 지원하는 경우, 다크 모드용 이미지를 별도로 등록할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **지원 포맷** | 배너, 네이티브, 팝업, 스플래시 (동영상 제외) |
| **노출 방식** | 사용자의 기기 테마 설정에 따라 자동 선택 |
| **폴백 동작** | 다크 모드 소재가 없으면 라이트 모드 소재 노출 |
| **심사** | 라이트/다크 소재가 동시에 심사되며, 하나라도 정책 위반 시 전체 거절 |

> 💡 다크 모드 소재 등록 여부는 매체사의 광고 유닛 설정에 따라 결정됩니다. 다크 모드 입력 필드가 표시되지 않으면 매체사가 다크 모드를 지원하지 않는 것입니다.

**동영상 소재**

| 항목 | 규격 |
|------|------|
| **포맷** | MP4 (권장), MOV, WebM, MKV, AVI |
| **최대 용량** | 1 GB |
| **영상 길이** | 5초 ~ 90초 |
| **최소 해상도** | 1080p |
| **비율** | 16:9 (가로) 또는 9:16 (세로) |

동영상 소재에는 커버 이미지가 필요합니다. 동영상 로딩 중 또는 재생 전에 표시됩니다.

| 항목 | 규격 |
|------|------|
| **포맷** | JPG, PNG, WebP |
| **비율** | 동영상과 동일 |

> 💡 iOS에서는 하단 34dp가 Safe Area입니다. 중요한 텍스트나 로고 배치를 피하세요.

> 💡 동영상은 업로드 후 여러 화질로 자동 변환됩니다. 영상 길이에 따라 변환에 몇 분이 소요될 수 있으니 여유 있게 등록하세요.

#### 소재 이름

소재를 식별하기 위한 이름입니다. 소재 목록에서 구분하기 쉬운 이름을 입력하세요.

#### 도착 URL

광고 클릭 시 이동할 페이지 주소를 입력합니다. 입력한 URL은 그대로 사용되므로 트래킹 파라미터나 딥링크를 자유롭게 활용할 수 있습니다.

| 유형 | 예시 | 설명 |
|------|------|------|
| **웹 URL** | `https://example.com/landing` | 일반적인 랜딩 페이지 |
| **트래킹 URL** | `https://tracker.example.com/click?campaign_id=123&redirect=...` | 전환 트래커가 포함된 URL |
| **딥링크** | `myapp://product/123` | 앱 내 특정 화면으로 이동 |
| **유니버설 링크** | `https://example.com/app/product/123` | iOS/Android 공통 딥링크 |
| **앱 스토어** | `https://play.google.com/store/apps/details?id=...` | 앱 설치 유도 |

> 💡 **딥링크 사용 시**: Android와 iOS를 별도로 구분할 필요 없이 유니버설 링크 또는 앱링크를 사용하면 플랫폼에 관계없이 동작합니다. 앱이 설치되지 않은 경우의 폴백(fallback) 처리는 딥링크 서비스 제공업체에서 설정하세요.

> 💡 **전환 추적 시**: AppsFlyer, Adjust, Branch 등의 트래킹 링크를 그대로 입력할 수 있습니다. URL은 변형 없이 그대로 전달됩니다.

### 네이티브 광고 추가 항목

네이티브 광고는 앱의 콘텐츠와 자연스럽게 어울리도록 구성됩니다. 매체사가 설정한 광고 단위에 따라 아래 항목이 표시됩니다.

#### CTA 버튼

행동 유도 버튼입니다. 매체사가 CTA 버튼을 활성화한 경우에만 표시됩니다.

"자세히 보기", "구매하기", "설치하기" 등 광고 목적에 맞는 버튼 텍스트를 선택할 수 있습니다. 버튼 배경색과 텍스트 색상도 직접 설정할 수 있습니다.

#### 광고주 프로필

광고주 정보를 표시합니다. 매체사가 프로필 표시를 활성화한 경우에만 표시됩니다.

| 항목 | 설명 |
|------|------|
| **프로필 이미지** | 256×256px 정사각형 이미지 |
| **프로필 이름** | 광고주 또는 브랜드 이름 (최대 20자) |
| **프로필 링크** | 프로필 클릭 시 이동할 URL |

#### 텍스트

광고에 표시될 텍스트입니다. 매체사가 설정한 텍스트 필드가 표시됩니다.

| 항목 | 설명 |
|------|------|
| **헤드라인** | 광고 제목 |
| **본문** | 광고 설명 |
| **추가 텍스트** | 매체사가 정의한 추가 필드 (선택) |

각 텍스트 필드에는 최대 글자 수가 설정되어 있습니다.

### 고급 설정 (선택)

캠페인 내에서 소재별로 다른 노출 일정을 설정할 수 있습니다. 예를 들어, 주중과 주말에 다른 소재를 노출하거나, 특정 기간에만 프로모션 소재를 노출할 수 있습니다.

> 💡 고급 설정은 매체사가 허용한 경우에만 표시됩니다. 스플래시 광고에서는 사용할 수 없습니다.

**노출 기간**

| 옵션 | 설명 |
|------|------|
| **캠페인 기간 전체** | 캠페인 시작일부터 종료일까지 노출됩니다. (기본값) |
| **기간 직접 설정** | 캠페인 기간 내에서 시작일/종료일과 시간을 직접 선택합니다. |

**노출 시간**

| 옵션 | 설명 |
|------|------|
| **항상 노출** | 24시간 내내 노출됩니다. (기본값) |
| **시간 지정** | 모든 요일에 동일한 시간대를 설정합니다. 예: 매일 09:00~18:00 |
| **요일별 지정** | 요일마다 다른 시간대를 설정합니다. 예: 평일 09:00~18:00, 주말 10:00~22:00 |

> 💡 시간대는 캠페인에 설정된 매체사 시간대를 기준으로 합니다.

## FAQ

**Q. 소재 심사는 얼마나 걸리나요?**

일반적으로 1~3 영업일 내에 심사가 완료됩니다. 심사 결과는 이메일로 알림이 발송됩니다.

**Q. 소재를 여러 개 등록할 수 있나요?**

네, 하나의 캠페인에 여러 소재를 등록할 수 있습니다. 승인된 소재들이 번갈아 노출됩니다.

**Q. 소재가 거절되면 어떻게 하나요?**

거절 사유를 확인하고 소재를 수정한 후 다시 제출하세요. 수정된 소재는 다시 심사를 받게 됩니다.

**Q. 동영상 업로드 후 바로 사용할 수 있나요?**

동영상은 업로드 후 여러 화질로 자동 변환됩니다. 영상 길이에 따라 변환에 몇 분이 소요될 수 있으니 여유 있게 등록하세요.
