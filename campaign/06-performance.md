# 성과 확인하기

캠페인 성과를 확인하고 리포트를 다운로드하는 방법을 설명합니다.

---

## 대시보드

대시보드에서 전체 캠페인의 성과를 한눈에 확인할 수 있습니다.

### 기간 선택

화면 상단의 날짜 선택기로 조회 기간을 설정합니다. 기본값은 최근 30일이며, 최대 1년까지 조회할 수 있습니다. 사전 설정된 기간(오늘, 어제, 최근 7일, 최근 30일)을 선택하거나 직접 시작일과 종료일을 지정할 수 있습니다.

### 성과 요약

선택한 기간의 전체 성과가 표시됩니다.

| 지표 | 설명 |
|------|------|
| **총 노출수** | 광고가 사용자에게 표시된 횟수 |
| **총 클릭수** | 사용자가 광고를 클릭한 횟수 |
| **평균 CTR** | 클릭률 (클릭수 ÷ 노출수 × 100) |

### 캠페인 상태

현재 진행 중인 캠페인 수와 상태별 현황을 확인할 수 있습니다.

---

## 리포트

사이드바의 **리포트** 메뉴에서 전체 캠페인의 성과를 상세하게 조회할 수 있습니다.

### 기간 선택

화면 상단에서 조회 기간을 설정합니다. 사전 설정된 기간을 선택하거나 날짜 선택기에서 직접 시작일과 종료일을 지정할 수 있습니다. 최대 1년까지 조회 가능합니다.

### 탭

상단 탭에서 조회할 데이터 유형을 선택합니다.

| 탭 | 설명 |
|------|------|
| **캠페인** | 캠페인별 성과 |
| **광고 상품** | 광고 상품별 성과 |
| **광고 유닛** | 광고 유닛별 성과 |
| **소재** | 소재별 성과 |

### 세그먼트

데이터를 더 세분화하여 볼 수 있습니다. 예를 들어, 캠페인 탭에서 **소재** 세그먼트를 선택하면 각 캠페인의 소재별 성과를 확인할 수 있습니다.

### 일별/시간별 조회

체크박스를 선택하여 데이터를 일별 또는 시간별로 분리해서 볼 수 있습니다.

- **일별**: 날짜별로 데이터를 분리하여 표시
- **시간별**: 시간대별로 데이터를 분리하여 표시

### 필터

특정 캠페인, 광고 상품, 광고 유닛, 소재만 조회할 수 있습니다. 필터를 추가하면 해당 항목만 테이블에 표시됩니다.

### 열 편집

**열 편집** 버튼을 클릭하면 테이블에 표시할 열을 선택할 수 있습니다. 드래그 앤 드롭으로 열 순서를 변경할 수도 있습니다. 변경한 설정은 자동으로 저장되어 다음 방문 시에도 유지됩니다.

### 성과 지표

리포트에서 확인할 수 있는 주요 지표입니다.

| 지표 | 설명 |
|------|------|
| **노출수** | 광고가 표시된 횟수 |
| **클릭수** | 광고가 클릭된 횟수 |
| **CTR** | 클릭률 (클릭수 ÷ 노출수 × 100) |
| **비용** | 광고비 |

테이블 하단에 선택한 기간의 합계가 표시됩니다.

### 정렬

열 제목을 클릭하면 해당 열을 기준으로 데이터를 정렬할 수 있습니다. 다시 클릭하면 정렬 방향이 변경됩니다.

### 다운로드

**CSV 내보내기** 버튼을 클릭하면 현재 조회 중인 데이터를 CSV 파일로 다운로드할 수 있습니다. 내보내기를 요청하면 파일이 준비되는 대로 이메일로 다운로드 링크가 발송됩니다.

## 캠페인 상세 성과

캠페인 목록에서 캠페인을 선택하면 상세 성과를 확인할 수 있습니다.

### 성과 카드

캠페인 상세 화면에서 다음 지표를 확인할 수 있습니다.

| 지표 | 설명 |
|------|------|
| **총 노출수** | 캠페인 시작부터 현재까지 누적 노출수 |
| **일 노출수** | 오늘 발생한 노출수 |
| **총 클릭수** | 캠페인 시작부터 현재까지 누적 클릭수 |
| **일 클릭수** | 오늘 발생한 클릭수 |

CPM 캠페인은 노출 목표 대비 달성률이, CPC 캠페인은 클릭 목표 대비 달성률이 진행 바로 표시됩니다.

### 예산 소진

캠페인 예산이 얼마나 소진되었는지 확인할 수 있습니다. 소진 금액과 잔여 예산, 소진율이 표시됩니다.

### 성과 리포트

**성과 리포트** 버튼을 클릭하면 해당 캠페인의 일별 상세 데이터를 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **날짜** | 데이터 기준일 |
| **노출수** | 해당 일자의 노출수 |
| **클릭수** | 해당 일자의 클릭수 |
| **CTR** | 해당 일자의 클릭률 |
| **비용** | 해당 일자의 광고비 (예상 금액) |

상단 선택기에서 캠페인 전체 또는 개별 소재를 선택하여 성과를 비교할 수 있습니다. **CSV** 또는 **Excel** 버튼으로 데이터를 다운로드할 수 있습니다.

---

## 정산서

캠페인이 종료되거나 취소되면 정산서를 확인할 수 있습니다. 캠페인 상세 화면에서 **정산서** 버튼을 클릭하면 새 창에서 정산서가 열립니다.

---

## 성과 지표 용어

| 용어 | 설명 |
|------|------|
| **노출수 (Impressions)** | 광고가 사용자 화면에 표시된 횟수 |
| **클릭수 (Clicks)** | 사용자가 광고를 클릭한 횟수 |
| **CTR (Click-Through Rate)** | 클릭률. 클릭수 ÷ 노출수 × 100 |
| **일 할당량** | 하루에 목표로 하는 노출수 또는 클릭수 |
| **달성률** | 목표 대비 현재 성과 비율 |
