# 캠페인 예약하기

광고 상품을 선택하고 캠페인을 예약하는 방법을 설명합니다.

[▶ 영상 보기](./assets/campaign-booking.mp4)

---

## 예약 단계

| 단계 | 이름 | 내용 |
|------|------|------|
| 1 | 광고 상품 선택 | 광고 상품, 기간, 타겟팅 |
| 2 | 캠페인 정보 | 캠페인 이름, 예산 |
| 3 | 결제 정보 | 담당자 정보, 결제 프로필 |
| 4 | 확인 | 예약 내용 확인 및 제출 |

---

## 1단계: 광고 상품 선택

### 광고 상품

1. **캠페인 예약** 버튼을 클릭합니다.
2. 광고 상품 목록에서 원하는 광고 상품을 선택합니다.

| 항목 | 설명 |
|------|------|
| **광고 상품명** | 광고 상품 이름 |
| **과금 유형** | CPM, CPC, CPP 중 하나 |
| **단가** | 과금 유형에 따른 비용 |

**과금 유형**

| 유형 | 설명 |
|------|------|
| **CPM** | Cost Per Mille. 광고가 1,000회 노출될 때마다 비용이 청구됩니다. |
| **CPC** | Cost Per Click. 사용자가 광고를 클릭할 때마다 비용이 청구됩니다. |
| **CPP** | Cost Per Period. 기간 단위로 고정 비용이 청구됩니다. 특정 기간 동안 광고 지면을 확보하고 싶을 때 적합합니다. |

### 기간 설정

시작일과 종료일을 선택합니다.

| 항목 | 규칙 |
|------|------|
| **최대 시작일** | 예약일로부터 약 5개월 후 |
| **최대 캠페인 기간** | 시작일로부터 최대 약 3개월 |

> 💡 시간대는 매체사가 설정한 기준을 따릅니다. 캠페인 기간 옆에 표시된 시간대를 확인하세요.

**CPP 광고 상품**

| 항목 | 설명 |
|------|------|
| **최소 구매 기간** | 광고 상품에 따라 최소 구매 기간이 설정되어 있을 수 있습니다. |
| **슬롯** | 동일 기간에 여러 광고주가 노출될 수 있습니다. 광고 상품에 설정된 슬롯 수를 확인하세요. |

> ⚠️ 시작일까지 7일 이내인 경우 **임박 예약**으로 분류됩니다. 매체사 상황에 따라 거절될 수 있으므로, 매체사에 직접 연락하여 심사를 요청하는 것을 권장합니다.

### 타겟팅 선택

타겟팅을 설정하면 전체가 아닌 특정 대상에게만 광고를 노출할 수 있습니다. 매체사가 제공하는 타겟팅 옵션 중 원하는 카테고리를 선택하세요.

**타겟팅 유형**

| 유형 | 설명 | 예시 |
|------|------|------|
| **오디언스 타겟팅** | **누구에게** 광고를 보여줄지 선택합니다. 사용자의 속성을 기반으로 특정 그룹에게만 노출됩니다. | 20대 여성, 게임 관심사 보유자 |
| **문맥 타겟팅** | **어디에** 광고를 보여줄지 선택합니다. 사용자가 보고 있는 콘텐츠의 주제를 기반으로 특정 지면에만 노출됩니다. | 스포츠 기사 페이지, 금융 뉴스 섹션 |

**필수 포함 / 필수 제외**

매체사가 광고 상품별로 타겟팅 옵션에 필수 포함 또는 필수 제외를 미리 설정해둘 수 있습니다.

| 옵션 | 설명 |
|------|------|
| **필수 포함** | 해당 카테고리에만 광고가 노출됩니다. 예: "스포츠" 필수 포함 시, 스포츠 관련 지면에서만 노출 |
| **필수 제외** | 해당 카테고리에는 광고가 노출되지 않습니다. 예: "정치" 필수 제외 시, 정치 관련 지면에서는 노출 안 됨 |

> 💡 필수 포함/제외는 매체사가 설정한 값이며, 광고주가 변경할 수 없습니다.

**할증률**

타겟팅을 선택하면 광고 단가에 할증률이 적용됩니다. 여러 타겟팅을 선택하면 할증률이 누적됩니다.

| 예시 | 계산 |
|------|------|
| 기본 CPM | 10,000원 |
| 연령대 타겟팅 | +10% |
| 성별 타겟팅 | +20% |
| **최종 CPM** | **13,000원** (10,000 × 1.3) |

> 💡 타겟팅을 설정하면 광고가 노출되는 대상이 좁아지므로 예상 노출수와 클릭수가 감소할 수 있습니다.

## 2단계: 캠페인 정보

### 캠페인 이름

캠페인을 식별할 수 있는 이름을 입력합니다.

### 예산 설정

**CPM/CPC 광고 상품**: 총 예산을 입력합니다. 일 할당량은 기간에 따라 자동으로 균등 분배됩니다.

**CPP 광고 상품**: 기간에 따른 고정 금액이 적용됩니다.

### 예상 성과

예산을 입력하면 화면 오른쪽에 예상 성과가 표시됩니다.

| 항목 | 설명 |
|------|------|
| **예산** | 입력한 총 예산 금액 |
| **사용 가능 금액** | 타겟팅 할증 적용 후 실제 광고 노출에 사용되는 금액 |
| **예상 노출수** | 예산으로 예상되는 광고 노출 횟수 |
| **예상 클릭수** | 예상되는 클릭 횟수 |
| **예상 CTR** | 예상 클릭률 |

> 💡 예상 성과는 과거 데이터를 기반으로 한 추정치이며, 실제 성과는 다를 수 있습니다.

## 3단계: 결제 정보

### 담당자 정보

| 항목 | 설명 |
|------|------|
| **담당자명** | 캠페인 담당자 이름 |
| **이메일** | 연락 가능한 이메일 주소 |

### 결제 수단

| 유형 | 설명 | 결제 시점 |
|------|------|----------|
| **카드 결제** | 등록된 카드로 선불 결제합니다. | 캠페인 제출 시 즉시 결제 |
| **수동 결제** | 매체사에 직접 연락하여 결제를 진행합니다. | 캠페인 승인 후 별도 안내 |

> 💡 매체사 설정에 따라 사용 가능한 결제 수단이 다를 수 있습니다. 카드 결제는 결제 프로필에 카드가 등록되어 있어야 사용할 수 있습니다.

### 사업자 정보

세금계산서 발행에 사용할 사업자 정보입니다. 광고 계정에 등록된 결제 프로필이 표시됩니다. 결제 프로필이 없는 경우 [광고 계정 만들기](./ad-account.md)를 참고하세요.

## 4단계: 확인

이전 단계에서 설정한 내용을 최종 확인합니다.

### 캠페인 정보

| 항목 | 내용 |
|------|------|
| **광고 상품** | 선택한 광고 상품명, 할인율 |
| **과금 유형** | 과금 유형 및 단가 (할증률 적용 시 변경된 단가 표시) |
| **캠페인 기간** | 시작일 ~ 종료일 |
| **국가** | 광고 노출 국가 |
| **광고주** | 광고주 이름 및 링크 |
| **계약 담당자** | 담당자명 및 이메일 |
| **결제 수단** | 카드 결제 또는 수동 결제 |
| **사업자 정보** | 사업자명, 사업자등록번호, 이메일 |
| **타겟팅** | 선택한 타겟팅 카테고리 및 할증률 |

### 예약 금액

| 항목 | 설명 |
|------|------|
| **예약 금액** | 총 예산 금액 (VAT 별도) |
| **예상 노출수** | 예상되는 광고 노출 횟수 |
| **예상 클릭수** | 예상되는 클릭 횟수 |

### 예약 약관 동의

예약을 제출하려면 예약 약관에 동의해야 합니다. **약관 보기** 버튼을 클릭하여 약관 내용을 확인할 수 있습니다.

### 임시 저장

예약을 완료하기 전에 임시 저장할 수 있습니다. 임시 저장된 캠페인은 **초안** 상태로 표시됩니다.

## 제출 후 진행 과정

### 카드 결제

| 단계 | 설명 |
|------|------|
| **1. 제출** | 캠페인을 제출하면 즉시 결제가 진행됩니다 |
| **2. 결제 완료** | 결제 성공 시 캠페인이 심사 대기 상태로 전환됩니다 |
| **3. 심사** | 매체사가 캠페인을 검토합니다 |
| **4. 승인/거절** | 거절 시 전액 환불됩니다 |

> ⚠️ 카드 한도 초과, 유효기간 만료 등으로 결제가 실패할 수 있습니다. 다른 카드를 선택하거나 카드 정보를 업데이트한 후 다시 시도하세요.

### 수동 결제

| 단계 | 설명 |
|------|------|
| **1. 제출** | 결제 없이 캠페인을 제출합니다 |
| **2. 심사** | 매체사가 캠페인을 검토합니다 |
| **3. 승인** | 승인 후 캠페인이 진행됩니다 |
| **4. 정산** | 매체사와 직접 정산을 진행합니다 |

### 심사 결과

| 결과 | 설명 |
|------|------|
| **승인** | 캠페인 상태가 **예약됨**으로 변경됩니다. 시작일이 되면 자동으로 **진행중**으로 전환됩니다. |
| **거절** | 거절 사유가 안내됩니다. 카드 결제의 경우 전액 환불됩니다. 캠페인을 수정한 후 다시 제출할 수 있습니다. |

> 💡 소재가 등록되지 않은 경우 캠페인이 승인되어도 광고 노출이 시작되지 않습니다.

## FAQ

**Q. 타겟팅을 선택하면 비용이 더 드나요?**

네, 타겟팅마다 할증률이 적용됩니다. 여러 타겟팅을 선택하면 할증률이 누적됩니다. 예약 화면에서 최종 단가를 확인할 수 있습니다.

**Q. 최소 예산이 있나요?**

매체사가 광고 상품별로 최소 예산을 설정할 수 있습니다. 예약 화면에서 최소 예산을 확인하세요.

**Q. 시작일이 얼마 남지 않았는데 예약할 수 있나요?**

시작일까지 7일 이내인 경우 임박 예약으로 분류되어 매체사 상황에 따라 거절될 수 있습니다. 급한 일정이라면 매체사에 직접 연락하여 빠른 심사를 요청하세요.

**Q. 카드 결제와 수동 결제의 차이는 무엇인가요?**

카드 결제는 제출 시 즉시 선불 결제되고, 캠페인 종료 후 자동으로 정산되어 미사용 금액이 환불됩니다. 수동 결제는 캠페인 종료 후 실제 사용 금액만 매체사와 직접 정산합니다.
