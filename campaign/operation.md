# 캠페인 운영하기

캠페인과 소재의 상태 확인 및 관리 방법을 설명합니다.

---

## 캠페인 상태

캠페인 목록 또는 상세 화면에서 현재 상태를 확인할 수 있습니다.

광고가 노출되려면 **캠페인**과 **소재**가 모두 승인되어야 합니다. 캠페인이 승인되었더라도 소재가 승인되지 않으면 광고가 노출되지 않으므로, 캠페인 승인 후 반드시 소재를 등록하고 심사를 받으세요.

**기본 흐름**: 작성중 → 심사중 → 예약됨 → 진행중 → 종료됨

| 상태 | 설명 | 다음 상태 |
|------|------|----------|
| **작성중** | 캠페인 정보 입력 중 | 제출 → 심사중 |
| **심사중** | 매체사 검토 대기 | 승인 → 예약됨 / 거절 → 거절됨 |
| **거절됨** | 심사 거절됨 | 수정 후 다시 제출 → 심사중 |
| **예약됨** | 심사 승인, 시작일 대기 | 시작일 되면 → 진행중 |
| **진행중** | 광고 노출 중 | 종료일 되면 → 종료됨 |
| **일시정지** | 예약됨/진행중에서 일시 중단 | 재개 → 이전 상태 |
| **취소됨** | 캠페인 취소됨 | - |
| **종료됨** | 캠페인 종료됨 | - |

---

## 캠페인 수정

캠페인이 작성중이거나 거절됨 상태일 때만 수정할 수 있습니다. 심사중인 경우 취소 후 다시 작성해야 하며, 승인된 후에는 광고주가 직접 수정할 수 없습니다. 수정이 필요하면 매체사에 문의하세요.

**승인 후 수정 불가 항목**: 광고 상품, 예산, 과금 유형

- 수정 가능: 작성중, 거절됨
- 수정 불가: 심사중, 예약됨, 진행중, 일시정지, 취소됨, 종료됨

---

## 캠페인 일시정지/재개

예약됨 또는 진행중인 캠페인을 일시정지할 수 있습니다. 일시정지하면 광고 노출과 예산 소진이 즉시 중단되며, 언제든 재개할 수 있습니다. 단, 일시정지 중에도 캠페인 종료일은 변경되지 않으므로 종료일이 지나면 캠페인은 자동으로 종료됩니다.

> ⚠️ 일시정지로 인해 미사용된 예산이 자동 환불되지 않을 수 있습니다. 일시정지 전 매체사와 협의하세요.

- 일시정지 가능: 예약됨, 진행중
- 일시정지 시: 광고 노출 중단, 예산 소진 중단, 종료일 유지

---

## 캠페인 취소

작성중, 심사중, 거절됨 상태의 캠페인만 취소할 수 있습니다. 심사중에 취소하면 전액 환불됩니다. 승인된 후(예약됨, 진행중, 일시정지)에는 광고주가 직접 취소할 수 없으며, 필요한 경우 매체사에 문의하세요. 취소된 캠페인은 복구할 수 없습니다.

- 취소 가능: 작성중, 심사중, 거절됨
- 취소 불가: 예약됨, 진행중, 일시정지 (매체사 문의)

---

## 캠페인 복제

기존 캠페인과 동일한 설정으로 새 캠페인을 빠르게 생성할 수 있습니다.

### 복제 방법

1. 캠페인 상세 화면에서 메뉴(⋯)를 클릭합니다.
2. **복제**를 선택합니다.
3. 복제된 캠페인이 **작성중** 상태로 생성됩니다.
4. 예산 등 필요한 항목을 설정하고 제출합니다.

### 복제되는 항목

| 항목 | 복제 여부 |
|------|----------|
| 광고 상품 | O |
| 타겟팅 | O (유효한 경우) |
| 노출 빈도 제한 | O |
| 결제 유형 | O |
| 담당자 정보 | O |
| 소재 | O (유효한 소재만, 재심사 필요) |

> ⚠️ 원본 캠페인의 타겟팅에 포함된 카테고리나 그룹이 삭제/비활성화된 경우, 해당 타겟팅은 초기화됩니다. 복제 후 타겟팅 설정을 반드시 확인하세요.

### 초기화되는 항목

| 항목 | 복제 후 값 |
|------|----------|
| 예산 | 0 (새로 설정 필요) |
| 시작일 | 현재로부터 7일 후 |
| 종료일 | 시작일 + 원본 캠페인 기간 |
| 제목 | 원본 제목 + 고유 ID |
| 상태 | 작성중 |
| 소재 상태 | 심사중 (재심사 필요) |

### 소재 복제 규칙

- 거절됨, 삭제됨 상태의 소재는 복제되지 않습니다.
- 복제된 소재는 모두 **심사중** 상태로 변경되어 재심사가 필요합니다.
- 광고 유닛이 변경되어 호환되지 않는 소재는 제외됩니다.

### 복제 불가 조건

다음 경우에는 캠페인을 복제할 수 없습니다.

- 광고 상품이 삭제된 경우
- 광고 상품의 유닛 세트가 없거나 수정이 필요한 경우

## 소재 상태

| 상태 | 설명 |
|------|------|
| **심사중** | 소재 심사 대기 |
| **승인됨** | 광고 노출 준비 완료 |
| **거절됨** | 소재 심사 거절, 수정 필요 |
| **일시정지** | 광고주가 노출 중지 |

소재가 거절되면 상세 화면에서 거절 사유를 확인할 수 있습니다. 사유에 따라 소재를 수정하고 다시 제출하세요. 자세한 심사 기준은 [광고 정책](./policy.md)을 참고하세요.

---

## 소재 수정

캠페인이 취소되거나 종료되기 전까지 소재를 수정할 수 있습니다. 소재 내용(파일, 도착 URL, 텍스트 등)을 변경하면 다시 심사를 받아야 하며, 심사 중에는 해당 소재가 노출되지 않습니다. 소재 이름만 변경하는 경우 심사 없이 즉시 반영됩니다.

- 수정 가능: 캠페인이 취소됨/종료됨 상태가 아닐 때
- 내용 변경: 재심사 필요, 심사 중 노출 중단
- 이름만 변경: 재심사 없이 즉시 반영

---

## 소재 삭제

캠페인이 취소되거나 종료되기 전까지 소재를 삭제할 수 있습니다. 삭제된 소재는 복구할 수 없으므로 신중하게 진행하세요.

- 삭제 가능: 캠페인이 취소됨/종료됨 상태가 아닐 때
- 삭제 후: 복구 불가

---

## 소재 일시정지/재개

승인된 소재를 일시정지하면 해당 소재의 노출이 중지됩니다. 다른 승인된 소재가 있으면 해당 소재가 대신 노출되며, 캠페인 예산 소진에는 영향이 없습니다. 일시정지된 소재는 언제든 재개할 수 있습니다.

- 일시정지 가능: 승인됨 상태의 소재
- 일시정지 시: 해당 소재만 노출 중단, 다른 소재는 계속 노출

---

## 환불

환불은 카드 결제 캠페인에만 적용됩니다. 수동 결제의 경우 매체사와 직접 정산합니다.

- 캠페인 심사 거절: 전액 환불
- 광고주가 취소 (심사중): 전액 환불
- 매체사가 취소: 미사용 예산 환불

---

캠페인과 소재의 주요 상태 변경 시 이메일로 알림을 받습니다.

| 대상 | 알림 발송 시점 |
|------|--------------|
| **캠페인** | 심사 승인, 심사 거절, 캠페인 시작, 캠페인 종료 |
| **소재** | 심사 승인, 심사 거절 |

## 정산

캠페인이 종료되거나 취소되면 정산이 진행됩니다. 결제 방식에 따라 정산 절차가 다릅니다.

### 카드 결제 (자동 정산)

카드로 결제한 캠페인은 자동으로 정산됩니다.

| 단계 | 설명 |
|------|------|
| **1. 캠페인 종료** | 캠페인이 종료되면 최종 성과가 집계됩니다 |
| **2. 정산서 발행** | 실제 사용 금액 기준으로 정산서가 자동 생성됩니다 |
| **3. 미사용 예산 환불** | 미사용 예산이 있으면 자동으로 환불됩니다 |

> 💡 카드 결제는 선불 방식입니다. 캠페인 제출 시 전체 예산이 결제되고, 종료 후 미사용 금액이 환불됩니다.

### 수동 결제 (직접 정산)

수동 결제를 선택한 캠페인은 매체사와 직접 정산합니다.

| 단계 | 설명 |
|------|------|
| **1. 캠페인 종료** | 캠페인이 종료되면 최종 성과가 집계됩니다 |
| **2. 정산서 발행** | 실제 사용 금액 기준으로 정산서가 생성됩니다 |
| **3. 직접 결제** | 매체사 안내에 따라 계좌이체 등으로 결제합니다 |

> 💡 수동 결제는 후불 방식입니다. 캠페인 종료 후 실제 사용 금액만 결제합니다. 세금계산서가 필요한 경우 매체사에 직접 요청하세요.

### 정산서 확인

캠페인 상세 화면에서 **정산서** 버튼을 클릭하면 정산서를 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **예약 금액** | 캠페인 예약 시 설정한 총 예산 |
| **사용 금액** | 실제 광고 노출/클릭으로 소진된 금액 |
| **환불 금액** | 미사용 예산 (카드 결제만 해당) |
| **세금** | VAT 등 적용 세금 |
| **최종 정산 금액** | 최종 청구/환불 금액 |

## FAQ

**Q. 캠페인을 일시정지하면 종료일도 연장되나요?**

아니요, 일시정지 중에도 캠페인 종료일은 변경되지 않습니다. 종료일이 지나면 캠페인은 자동으로 종료됩니다.

**Q. 승인된 캠페인을 취소할 수 있나요?**

승인된 후(예약됨, 진행중, 일시정지)에는 광고주가 직접 취소할 수 없습니다. 취소가 필요한 경우 매체사에 문의하세요. 위약금이 발생할 수 있습니다.

**Q. 미사용 예산은 자동으로 환불되나요?**

카드 결제의 경우 캠페인 종료 후 미사용 예산이 자동으로 환불됩니다. 수동 결제는 실제 사용 금액만 정산하므로 별도 환불 절차가 없습니다.
