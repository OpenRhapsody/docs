# 광고 정책

광고 심사 기준과 정책을 설명합니다.

---

## 심사 개요

캠페인과 소재는 매체사 심사를 거쳐 승인됩니다. 심사는 광고 품질과 사용자 경험을 보호하기 위해 진행됩니다.

### 심사 대상

| 대상 | 심사 내용 |
|------|----------|
| **캠페인** | 예약 일정, 예산, 브랜드 정책 |
| **소재** | 콘텐츠 품질, 기술 규격, 정책 준수 |

### 심사 결과

| 결과 | 설명 |
|------|------|
| **승인** | 광고 노출 가능 |
| **거절** | 수정 후 재제출 필요 (사유 안내) |

---

## 캠페인 거절 사유

### 일정 충돌

| 사유 | 설명 |
|------|------|
| **이미 예약된 일정** | 요청한 기간에 이미 다른 캠페인이 예약되어 있습니다. 일정을 조정하여 다시 제출하세요. |

### 정책 위반

| 사유 | 설명 |
|------|------|
| **브랜드 정책** | 매체사의 브랜드 정책에 맞지 않습니다. |

### 심사 지연

| 사유 | 설명 |
|------|------|
| **기한 내 심사 미완료** | 정해진 기한 내에 심사가 완료되지 않았습니다. |

---

## 소재 거절 사유

### 법률 및 규정

| 사유 | 설명 |
|------|------|
| **현행법 위반** | 현행법 및 주요 권고 사항을 위반하는 내용입니다. |
| **업종별 정책 미준수** | 규제 업종의 경우 심의 번호 제출 및 필수 문구 삽입이 필요합니다. |
| **광고 불가 업종** | 해당 업종은 광고가 허용되지 않습니다. |

### 사용자 보호

| 사유 | 설명 |
|------|------|
| **사용성 침해** | 사용자 경험을 해치는 콘텐츠입니다. |
| **청소년 보호** | 청소년에게 유해한 콘텐츠입니다. |

### 콘텐츠 품질

| 사유 | 설명 |
|------|------|
| **허위/과장/기만** | 허위, 과장, 또는 기만적인 내용을 포함합니다. |
| **비교/비방** | 경쟁사를 부당하게 비교하거나 비방하는 내용입니다. |

### 부적절한 콘텐츠

| 사유 | 설명 |
|------|------|
| **음란/선정** | 음란하거나 선정적인 콘텐츠입니다. |
| **폭력/혐오** | 폭력, 혐오, 공포, 비속어 등을 포함합니다. |
| **사회 정서 침해** | 보편적 사회 정서에 반하는 내용입니다. |

### 권리 침해

| 사유 | 설명 |
|------|------|
| **타인의 권리 침해** | 저작권, 초상권 등 타인의 권리를 침해합니다. |

### 기술적 문제

| 사유 | 설명 |
|------|------|
| **소재 변환 실패** | 소재 파일 처리 중 오류가 발생했습니다. |

---

## 금지 콘텐츠

다음 콘텐츠는 광고에 사용할 수 없습니다.

### 절대 금지

- 성인용/음란물 콘텐츠
- 폭력, 고어, 잔인한 콘텐츠
- 혐오 발언, 차별, 괴롭힘
- 저작권/지적재산권 침해
- 허위 건강 또는 재정 주장
- 근거 없는 제품 주장
- 미성년자 착취 또는 유해 콘텐츠

### 규제 업종 (조건부)

다음 업종은 추가 요건이 있을 수 있습니다:

| 업종 | 요건 |
|------|------|
| **주류** | 연령 제한, 경고 문구 |
| **담배** | 규제에 따라 제한 |
| **도박** | 라이선스 확인, 연령 제한 |
| **의약품** | 심의 번호, 필수 문구 |
| **건강식품** | 심의 번호, 필수 문구 |
| **금융 서비스** | 규제 준수 확인 |
| **다이어트/체중 감량** | 허위 주장 금지 |
| **암호화폐** | 규제 준수 확인 |

## 기술 요구사항

### 이미지 소재

| 항목 | 요구사항 |
|------|----------|
| **포맷** | JPG, PNG, WebP, GIF |
| **최대 용량** | 10 MB |
| **해상도** | 광고 규격의 3배 이상 |
| **비율** | 광고 규격과 동일 (1% 오차 허용) |

### 동영상 소재

| 항목 | 요구사항 |
|------|----------|
| **포맷** | MP4 (권장), MOV, WebM |
| **최대 용량** | 1 GB |
| **길이** | 5초 ~ 90초 |
| **해상도** | 1080p 이상 |
| **비율** | 16:9 또는 9:16 |

### 도착 URL

| 항목 | 요구사항 |
|------|----------|
| **유효성** | 정상 작동하는 URL |
| **보안** | HTTPS 권장 |
| **콘텐츠** | 광고 내용과 일관성 유지 |

## 업종별 추가 요건

규제 업종의 광고는 다음을 준수해야 합니다.

### 심의 번호

해당 업종의 경우 심의 번호를 소재에 표시해야 합니다:

- 의약품
- 건강기능식품
- 의료기기
- 금융 상품

### 필수 문구

법적으로 요구되는 경고 문구나 고지 사항을 포함해야 합니다.

---

## 거절 시 대응

### 거절 사유 확인

1. 캠페인 또는 소재 상세 화면에서 거절 사유를 확인합니다.
2. 이메일로도 거절 사유가 안내됩니다.

### 수정 및 재제출

1. 거절 사유에 따라 소재 또는 캠페인을 수정합니다.
2. **저장** 버튼을 클릭하여 다시 제출합니다.
3. 새로운 심사가 진행됩니다.

### 문의

거절 사유에 대해 문의가 있는 경우 매체사에 연락하세요.

---

## 정책 준수 팁

- 광고 내용과 도착 페이지의 내용이 일치해야 합니다.
- 과장되거나 검증되지 않은 주장을 피하세요.
- 저작권이 있는 이미지나 음악 사용 시 권리를 확보하세요.
- 규제 업종의 경우 필요한 심의 번호와 문구를 미리 준비하세요.
- 기술 규격을 확인하고 고품질 소재를 준비하세요.
