# 캠페인 예약하기 (광고주)

이 가이드에서는 광고주가 캠페인을 예약하고 소재를 등록하여 광고를 집행하는 과정을 설명합니다. 캠페인의 기본 개념부터 예약, 결제, 소재 등록까지 단계별로 안내합니다.

---

## 캠페인이란?

**캠페인**은 광고 집행의 기본 단위입니다. 광고주는 매체사가 판매하는 광고 상품을 선택하고, 원하는 기간과 예산을 설정하여 캠페인을 예약합니다. 캠페인이 승인되면 소재를 등록하고, 시작일이 되면 광고가 노출됩니다.

### 캠페인 운영 흐름

캠페인은 예약부터 광고 노출까지 다음 단계로 진행됩니다.

| 단계 | 담당 | 설명 |
|------|------|------|
| **1. 캠페인 예약** | 광고주 | 광고 상품 선택, 기간/예산 설정, 타겟팅 선택 |
| **2. 결제 및 제출** | 광고주 | 결제 정보 입력, 약관 동의 후 제출 |
| **3. 캠페인 심사** | 매체사 | 캠페인 내용 검토 후 승인/거절 |
| **4. 소재 등록** | 광고주 | 이미지/동영상 소재 업로드, 도착 URL 설정 |
| **5. 소재 심사** | 매체사 | 소재 내용 검토 후 승인/거절 |
| **6. 광고 노출** | 시스템 | 시작일부터 종료일까지 자동 노출 |

> 💡 소재는 캠페인 제출과 동시에 등록하거나, 캠페인 승인 후 별도로 등록할 수 있습니다. 캠페인과 소재가 모두 승인되어야 광고가 노출됩니다.

---

## 캠페인 구성 요소

### 광고 상품

매체사가 판매하는 광고 인벤토리입니다. 광고주는 광고 상품을 선택하여 캠페인을 예약합니다.

| 구성 | 설명 |
|------|------|
| **광고 단위** | 광고가 노출되는 지면 (배너, 전면, 네이티브 등) |
| **과금 유형** | CPM, CPC, CPP |
| **단가** | 노출/클릭/기간당 비용 |
| **타겟팅** | 특정 사용자나 콘텐츠에 광고를 노출하는 옵션 (연령대, 성별, 카테고리 등) |

**과금 유형**
- **CPM** (Cost Per Mille): 1,000회 노출당 비용
- **CPC** (Cost Per Click): 클릭당 비용
- **CPP** (Cost Per Period): 기간당 고정 비용

### 기간 및 예산

| 항목 | 설명 |
|------|------|
| **시작일/종료일** | 캠페인 진행 기간 |
| **총 예산** | 캠페인 전체 예산 |

### 소재

사용자에게 노출되는 광고 콘텐츠입니다.

| 항목 | 설명 |
|------|------|
| **파일** | 이미지(JPG, PNG, WebP, GIF) 또는 동영상(MP4, MOV, WebM) |
| **도착 URL** | 클릭 시 이동할 페이지 |
| **CTA** | 행동 유도 버튼 (자세히 보기, 구매하기 등) |
