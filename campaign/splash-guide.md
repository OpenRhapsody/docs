# 스플래시 광고 예약 가이드

앱 시작 시 전체 화면으로 노출되는 스플래시 광고 상품을 예약하는 방법을 설명합니다.

---

## 스플래시 광고란?

**스플래시 광고**는 사용자가 앱을 실행할 때 가장 먼저 보이는 전체 화면 광고입니다. 앱 로딩 중에 노출되어 높은 주목도를 확보할 수 있습니다. 클릭 기능이 없는 노출 전용 광고로, 브랜드 인지도 향상에 효과적입니다.

### 특징

- **전체 화면 노출**: 앱 시작 시 화면 전체에 표시
- **높은 주목도**: 사용자가 반드시 보게 되는 광고
- **노출 전용**: 클릭 기능 없음 (도착 URL 불필요)
- **브랜딩에 적합**: 브랜드 런칭, 시즌 캠페인에 효과적

### 활용 사례

- 신제품 출시 홍보
- 브랜드 인지도 향상
- 시즌 프로모션 (명절, 블랙프라이데이 등)
- 대규모 이벤트 알림

---

## 소재

스플래시 광고는 이미지만 지원합니다. 동영상과 GIF는 사용할 수 없습니다.

### 이미지 규격

| 항목 | 규격 |
|------|------|
| **포맷** | JPG, PNG, WebP |
| **비율** | 4:3 (360×270 기준) |
| **최소 해상도** | 1080×810 (3배) |
| **최대 용량** | 10 MB |

### 등록 방법

1. 캠페인 상세 화면에서 **소재 등록** 버튼을 클릭합니다.
2. 규격에 맞는 이미지를 업로드합니다.
3. 소재 이름을 입력합니다.
4. **제출** 버튼을 클릭합니다.

스플래시 광고는 클릭 기능이 없어 도착 URL, CTA 버튼은 입력하지 않습니다.

### 다크 모드

매체사가 다크 모드를 지원하는 경우, 다크 모드용 이미지를 별도로 등록할 수 있습니다.

### 노출 시간

광고가 자동으로 닫히는 시간은 매체사가 설정합니다. 일반적으로 0.5초~3초 사이입니다.

---

## 72시간 사전 확정 규칙

스플래시 광고는 사용자 기기에 미리 다운로드되어 앱 시작 시 즉시 노출됩니다. 이를 위해 캠페인 시작 **72시간 전**까지 소재와 일정이 확정되어야 합니다.

### 사전 확정이 필요한 이유

1. 캠페인 시작 72시간 전부터 소재가 사용자 기기에 미리 저장됩니다.
2. 저장된 정보로 앱 시작 시 즉시 광고가 노출됩니다.
3. 네트워크 상태와 관계없이 안정적으로 노출됩니다.

### 72시간 이내 변경 시 주의사항

캠페인 시작 72시간 이내에 변경하면 다음과 같은 문제가 발생할 수 있습니다.

| 변경 유형 | 영향 |
|------|------|
| **소재 변경** | 일부 사용자에게 기존 소재가 계속 노출됨 |
| **일정 변경/취소** | 일부 사용자에게 기존 일정대로 노출됨 |

새로운 정보는 사용자가 앱을 재시작한 후에만 반영됩니다. 기존 소재와 새 소재 노출 모두 과금 대상이며, 위약금은 매체사 정책에 따릅니다.

> ⚠️ 소재는 캠페인 시작 **72시간 이상 전**에 승인받는 것을 권장합니다.

## 성과 리포트

스플래시 광고는 클릭 기능이 없으므로 리포트에서 클릭 관련 지표는 표시되지 않습니다.

- 노출수만 집계됨
- 클릭수: 항상 0
- CTR: 해당 없음

## FAQ

**Q. 스플래시 광고는 클릭할 수 있나요?**

아니요, 스플래시 광고는 노출 전용 광고입니다. 클릭 기능이 없으므로 도착 URL을 설정하지 않습니다.

**Q. 72시간 사전 확정 규칙은 왜 있나요?**

스플래시 광고는 앱 시작 시 즉시 노출되도록 사용자 기기에 미리 다운로드됩니다. 72시간 전에 확정되어야 모든 사용자에게 안정적으로 노출됩니다.

**Q. 캠페인 시작 72시간 이내에 소재를 변경하면 어떻게 되나요?**

일부 사용자에게는 기존 소재가 계속 노출될 수 있습니다. 새 소재는 사용자가 앱을 재시작한 후에만 반영됩니다. 기존 소재와 새 소재 노출 모두 과금됩니다.
