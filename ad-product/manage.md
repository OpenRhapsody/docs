# 광고 상품 관리하기

생성된 광고 상품을 수정하거나 삭제하는 방법을 안내합니다.

---

## 광고 상품 상태

광고 상품 목록에서 각 상품의 현재 상태를 확인할 수 있습니다. 상태에 따라 광고주의 캠페인 예약 가능 여부가 달라집니다.

| 상태 | 설명 | 캠페인 예약 |
|------|------|:----------:|
| **준비중** | 유닛 세트에 아직 SDK 연동이 완료되지 않은 유닛이 있습니다 | ❌ |
| **판매중** | 모든 유닛이 SDK 연동을 완료하여 광고주가 캠페인을 예약할 수 있습니다 | ✅ |
| **삭제중** | 삭제 요청 후 완전히 삭제되기까지 대기 중입니다 | ❌ |

> 💡 **준비중 → 판매중 전환 조건**
>
> 유닛 세트에 포함된 모든 유닛에서 광고 요청이 1회 이상 발생해야 합니다. 앱에 SDK를 연동하고 광고 유닛이 노출되는 화면을 한 번 이상 띄우면 자동으로 판매중 상태로 전환됩니다.

---

## 수정하기

### 수정 절차

1. **광고 상품** 메뉴에서 수정할 상품을 선택합니다.
2. 상세 페이지 우측 상단의 **[...]** 버튼을 클릭합니다.
3. **수정**을 선택합니다.
4. 설정을 변경하고 **상품 만들기**를 클릭합니다.

### 수정 가능한 항목

| 항목 | 설명 |
|------|------|
| 광고 상품명 | 광고주에게 표시되는 상품 이름 |
| 과금 유형 | CPM, CPC, CPP |
| 판매 정보 | 단가, 상품 할인율, 최소 집행 금액/기간 |
| 판매 기준 지표 | 일 노출수, 예상 클릭률 |
| 타겟팅 설정 | 타겟팅 카테고리, 필수 선택 여부 |
| 고급 설정 | 노출 빈도 제한, 영상 스킵 딜레이 |
| 유닛 세트 구성 | 유닛 추가/제거, 세트 이름 변경 |

> 💡 **할인율 변경하기**
>
> 프로모션이나 시즌 이벤트에 맞춰 상품 할인율을 변경할 수 있습니다. 수정 화면의 **판매 정보** 섹션에서 할인율(%)을 입력하세요. 변경된 할인율은 새로 예약되는 캠페인부터 적용됩니다.

### 주의사항

> ⚠️ **수정 필요 유닛**
>
> 유닛 세트에 수정 필요 상태의 유닛이 포함되어 있으면 광고 상품을 수정할 수 없습니다. 먼저 해당 유닛을 세트에서 제거하거나, 유닛 설정을 원래대로 되돌리세요.

> ⚠️ **캠페인 영향**
>
> 수정 사항은 새 캠페인부터 적용됩니다. 진행 중인 캠페인은 예약 당시의 설정으로 운영됩니다.

## 삭제하기

### 삭제 절차

1. **광고 상품** 메뉴에서 삭제할 상품을 선택합니다.
2. 상세 페이지 우측 상단의 **[...]** 버튼을 클릭합니다.
3. **상품 삭제**를 선택합니다.
4. 삭제할 상품명을 입력하고 **삭제** 버튼을 클릭합니다.

### 주의사항

> ⚠️ **권한**
>
> 삭제는 **소유자(Owner)** 권한 이상이 필요합니다.

> ⚠️ **삭제 대기 기간**
>
> 삭제 요청 후 바로 삭제되지 않고 **삭제중** 상태가 됩니다.
> - 진행 중인 캠페인이 없으면 요청 후 3일 뒤 완전히 삭제
> - 진행 중인 캠페인이 있으면 마지막 캠페인 종료 후 3일 뒤 완전히 삭제
>
> 삭제중 상태에서는 새 캠페인 예약과 상품 수정이 불가능합니다.

### 삭제 취소하기

삭제중 상태의 상품은 완전히 삭제되기 전까지 복원할 수 있습니다.

1. **광고 상품** 메뉴에서 삭제중 상태의 상품을 선택합니다.
2. 상세 페이지 우측 상단의 **[...]** 버튼을 클릭합니다.
3. **삭제 취소**를 선택합니다.

> 💡 삭제가 취소되면 즉시 이전 상태(판매중 또는 준비중)로 복원됩니다.

> ⚠️ **복구 불가**
>
> 완전히 삭제된 광고 상품은 복구할 수 없습니다. 삭제 대기 기간 중에만 취소할 수 있습니다.

## 목록 관리

### 노출 순서 변경

광고주에게 표시되는 광고 상품의 순서를 변경할 수 있습니다.

1. **광고 상품** 메뉴에서 **순서 변경** 버튼을 클릭합니다.
2. 드래그 앤 드롭으로 상품 순서를 조정합니다.
3. **변경사항 저장**을 클릭합니다. (취소하려면 **취소** 버튼 클릭)

> 💡 인기 상품이나 추천 상품을 상단에 배치하면 광고주의 선택을 유도할 수 있습니다.

### 목록 필터링

광고 상품 목록에서 다양한 조건으로 필터링하여 원하는 상품을 빠르게 찾을 수 있습니다.

- **상태별 필터**: 준비중, 판매중, 삭제중
- **과금 유형별 필터**: CPM, CPC, CPP

### 타겟팅 정보 확인

상품 상세 페이지의 **상품 정보** 섹션에서 다음을 확인할 수 있습니다:

- 상품에 설정된 타겟팅 카테고리
- 필수 선택 카테고리
- 필수 타겟팅 오디언스 규모

## FAQ

**Q. 수정하면 진행 중인 캠페인에 영향이 있나요?**

없습니다. 수정 사항은 새로 예약되는 캠페인부터 적용됩니다. 진행 중인 캠페인은 예약 당시의 설정으로 운영됩니다.

**Q. 삭제 후 같은 이름으로 다시 만들 수 있나요?**

네, 삭제 완료 후에는 동일한 이름으로 새 광고 상품을 생성할 수 있습니다.

**Q. 준비중 상태에서 판매중으로 바뀌지 않아요.**

유닛 세트에 포함된 모든 유닛에서 광고 요청이 1회 이상 발생해야 합니다. 앱에 SDK를 연동하고, 광고 유닛이 노출되는 화면을 한 번 이상 띄워주세요.
