# 광고 상품 생성하기

광고 상품의 각 설정 항목을 상세히 안내합니다.

---

## 생성 화면 진입

1. 좌측 메뉴에서 **광고 상품**을 클릭합니다.
2. **+ 상품 생성** 버튼을 클릭합니다.

---

## 1단계: 상품 정보 입력

### 상품 정보

| 항목 | 설명 | 필수 |
|------|------|:----:|
| 광고 상품명 | 광고주에게 표시되는 상품 이름 | O |
| 과금 유형 | CPM, CPC, CPP 중 선택 | O |
| 비공개 | 광고주에게 노출하지 않고 매체사만 사용 | - |

#### 광고 상품명 작성 가이드

광고 상품명은 **광고 상품 소개서와 1:1로 대응**되도록 작성하세요. 광고주는 소개서를 보고 상품을 검토한 뒤 콘솔에서 캠페인을 예약합니다. 소개서와 콘솔의 상품명이 일치하면 광고주가 원하는 상품을 빠르게 찾아 예약할 수 있습니다.

- 소개서에 "홈 배너 CPM"이 있다면 콘솔에도 동일한 이름으로 생성
- 지면과 과금 유형이 명확히 드러나도록 작성
- 예시: `홈 배너 CPM`, `피드 네이티브 CPC`, `스플래시 CPP`

#### 과금 유형

| 유형 | 설명 | 적합한 상황 |
|------|------|-------------|
| **CPM** | 1,000회 노출당 과금 | 노출 보장이 필요한 광고, 안정적 수익 |
| **CPC** | 클릭당 과금 | 전환 중심 광고, 광고주 설득이 쉬움 |
| **CPP** | 기간 단위 과금 | 프리미엄 지면, 독점 노출 보장 |

### 판매 정보

| 항목 | 설명 |
|------|------|
| 통화 | 결제 통화 (KRW, JPY, INR) |
| 시간대 | 캠페인 시작/종료 기준 시간대 |
| 단가 | 상품 가격 |
| 상품 할인율 | 상품 할인율 (%) |

> 💡 **글로벌 지원**: 통화와 시간대를 설정하면 해외 광고주에게도 광고 상품을 판매할 수 있습니다.
>
> - 대한민국 (KRW, Asia/Seoul)
> - 일본 (JPY, Asia/Tokyo)
> - 인도 (INR, Asia/Kolkata)
>
> 추가 국가가 필요하면 문의해 주세요. ([contact@adrop.io](mailto:contact@adrop.io))

#### CPM 추가 항목

| 항목 | 설명 |
|------|------|
| 최소 집행 금액 | 캠페인 최소 예산 |
| 동영상 추가 단가 | 동영상 소재 사용 시 추가되는 단가 (동영상 지원 유닛 포함 시) |

#### CPC 추가 항목

| 항목 | 설명 |
|------|------|
| 최소 집행 금액 | 캠페인 최소 예산 |

#### CPP 추가 항목

| 항목 | 설명 |
|------|------|
| 최소 집행 기간 | 캠페인 최소 기간 (1~150일) |
| 구좌 수 | 동시에 판매할 수 있는 광고 수 (1~12) |

> 💡 **구좌 수란?**
>
> 같은 지면에 동시에 노출할 수 있는 광고 수입니다. 1구좌는 한 광고주가 독점 노출하고, 2구좌 이상은 여러 광고주의 광고가 캐러셀이나 랜덤 분배로 번갈아 노출됩니다.

### 판매 기준 지표

광고주에게 예상 성과를 안내하고, 캠페인 분배를 최적화하는 데 사용됩니다.

| 항목 | 설명 |
|------|------|
| 일 노출수 | 하루 기준, 이 광고 상품으로 판매 가능한 최대 노출수를 입력합니다. |
| 예상 클릭률 | 이 광고 상품의 평균 클릭률을 입력합니다. (클릭 수 ÷ 노출 수 × 100) |

> 💡 **일 노출수**는 일 최대 노출 한도로 동작하여 과금과 분배의 안정성을 보장합니다. 초기 설정 시 직접 입력이 필요하지만, 광고 데이터가 누적되면 자동으로 추세를 예측합니다. (추후 **인벤토리 인사이트** 애드온으로 제공 예정)

### 부가 정보

| 항목 | 설명 |
|------|------|
| 소재 제작 가이드 링크 | (선택) 광고주에게 제공할 소재 규격 안내 페이지 URL. 이미지 사이즈, 파일 형식, 텍스트 가이드라인 등을 정리한 문서를 연결하면 광고주가 소재를 빠르게 준비할 수 있습니다. |

### 고급 설정

| 항목 | 설명 |
|------|------|
| 노출 빈도 제한 | 하루 동안 한 사용자에게 이 광고 상품으로 예약된 캠페인이 최대 몇 번 노출될지 설정합니다. (기본값: 50) |
| 영상 스킵 딜레이 | (동영상 유닛 포함 시) 영상 광고 시작 후 몇 초 뒤에 스킵 버튼이 활성화될지 설정합니다. (기본값: 3초) |

> 💡 **노출 빈도 제한 설정 팁**
>
> 같은 광고를 너무 자주 보면 사용자는 피로감을 느끼고, 클릭률이 떨어집니다. 반면 너무 적게 노출하면 광고 효과가 줄어듭니다. 브랜딩 목적의 광고는 빈도를 높게, 전환 목적의 광고는 적절히 제한하는 것이 효과적입니다.

### 타겟팅

광고 상품에 타겟팅 옵션을 설정할 수 있습니다. 광고주가 선택할 수 있는 타겟팅 카테고리를 지정하고, 필수 선택 여부를 설정합니다.

👉 타겟팅에 대한 자세한 내용은 [타겟팅 설정](../targeting/index.md)을 참고하세요.

### 광고 유닛 선택

우측 유닛 목록에서 이 상품에 포함할 광고 유닛을 선택합니다.

- 최소 1개 이상의 유닛을 선택해야 합니다.
- 플랫폼별(Android/iOS)로 분리 등록을 권장합니다.

> ⚠️ **유닛 중복 사용 주의**
>
> 하나의 광고 유닛은 하나의 광고 상품에만 포함하는 것을 권장합니다. 같은 유닛이 여러 상품에 포함되면 인벤토리 계산이 복잡해지고, 캠페인 간 경쟁으로 예상과 다른 노출 결과가 발생할 수 있습니다. CPP 상품의 경우 구좌 수를 초과하여 예약을 받게 될 수 있습니다.

**다음** 버튼을 클릭합니다.

## 2단계: 유닛 세트 구성

[▶ 영상 보기](./assets/ad-product-unit-set.mp4)

선택한 유닛들을 세트로 구성합니다. 유닛 세트는 광고주가 소재를 업로드하는 단위이며, 세트 이름이 광고주에게 표시되므로 지면을 명확하게 구분할 수 있는 이름을 사용하세요.

### 유닛 세트란?

유닛 세트는 **광고주에게 소재를 받을 지면들의 묶음**입니다. 같은 세트에 포함된 유닛들은 동일한 소재를 공유하므로, 광고주가 유닛마다 소재를 따로 등록할 필요가 없습니다.

예를 들어 Android 홈 배너와 iOS 홈 배너를 같은 세트로 묶으면, 광고주는 소재 하나만 등록해도 양쪽 플랫폼에 광고가 노출됩니다.

### 자동 구성

같은 포맷, 사이즈, 옵션(템플릿, 워터마크, 캐러셀 등)을 가진 유닛들이 자동으로 하나의 세트로 묶입니다.

| 구성 기준 | 설명 |
|----------|------|
| 포맷 | 배너, 전면, 네이티브 등 광고 형식 |
| 사이즈 | 광고 크기 (예: 320x50, 300x250) |
| 템플릿 | 템플릿 사용 여부 |
| 워터마크 | 워터마크 지원 여부 |
| 캐러셀 | 캐러셀 지원 여부 |

### 유닛 세트 설정

각 유닛 세트의 이름과 유닛 배치를 설정합니다. 유닛 세트는 광고주가 소재를 등록하는 단위입니다.

| 항목 | 설명 |
|------|------|
| 세트 이름 | 광고주에게 표시되는 이름 (예: 홈 배너, 피드 배너) |
| 유닛 배치 | 드래그 앤 드롭으로 유닛을 세트 간 이동 |

> 💡 같은 유닛 세트 내 유닛은 동일한 소재를 공유합니다. 광고주가 소재 하나만 등록하면 세트 내 모든 유닛에 노출됩니다.

### 통합 관리 vs 분리 관리

같은 사이즈·형식의 유닛이라도 세트를 분리하여 각각 다른 소재를 받을 수 있습니다.

| 방식 | 설명 | 적합한 상황 |
|------|------|-------------|
| **통합 관리** | 여러 유닛을 하나의 세트로 | 소재 하나로 여러 지면에 노출할 때 |
| **분리 관리** | 유닛별로 세트 분리 | 사용자 피로도, UX를 고려해 지면별 다른 소재를 받고 싶을 때 |

### 주의사항

> ⚠️ **유닛 세트 미지정 시**
>
> 유닛 세트가 지정되지 않은 유닛이 있으면 광고 상품을 저장할 수 없습니다.

> ⚠️ **유닛 수정 시 영향**
>
> 광고 상품에 포함된 유닛의 포맷, 사이즈, 옵션(템플릿, 워터마크 등)을 수정하면 세트 기준과 맞지 않아 **수정 필요** 상태가 됩니다.
>
> 수정 필요 상태의 유닛이 포함된 광고 상품은:
> - 새 캠페인을 예약할 수 없습니다
> - 기존 캠페인은 정상 운영됩니다
> - 광고 상품 목록에서 경고 아이콘이 표시됩니다
>
> **해결 방법**: 광고 상품 수정 화면에서 해당 유닛을 세트에서 제거하거나, 유닛 설정을 원래대로 되돌리세요.

## 3단계: 상품 만들기

**상품 만들기** 버튼을 클릭하면 생성이 완료됩니다.

생성된 상품은 **광고 상품** 목록에서 확인할 수 있습니다.

| 상태 | 설명 |
|------|------|
| **Live** | 광고주가 캠페인 예약 가능 |
| **Paused** | 일시정지 상태, 예약 불가 |

> 💡 유닛 세트에 연결된 유닛이 모두 SDK 연동 상태여야 Live가 됩니다.

## FAQ

**Q. CPM, CPC, CPP 중 어떤 과금 유형을 선택해야 하나요?**

A. 노출 보장이 필요하면 CPM, 전환 중심 광고주가 많으면 CPC, 프리미엄 지면의 독점 노출을 판매하려면 CPP를 선택하세요.

**Q. 같은 유닛을 여러 광고 상품에 포함해도 되나요?**

A. 권장하지 않습니다. 같은 유닛이 여러 상품에 포함되면 인벤토리 계산이 복잡해지고, CPP 상품의 경우 구좌 수를 초과하여 예약을 받게 될 수 있습니다.

**Q. 유닛 세트 이름은 어떻게 지어야 하나요?**

A. 광고주가 소재를 업로드할 때 보는 이름입니다. "홈 배너", "피드 중간" 등 지면을 명확히 구분할 수 있는 이름을 사용하세요.

**Q. 일 노출수와 예상 클릭률은 어떻게 설정하나요?**

A. 일 노출수는 하루 최대 판매 가능한 노출수, 예상 클릭률은 평균 클릭률을 입력합니다. 초기에는 보수적으로 설정하고, 데이터가 쌓이면 조정하세요.

**Q. 해외 광고주에게도 판매할 수 있나요?**

A. 네, 판매 정보에서 통화와 시간대를 설정하면 해외 광고주에게도 광고 상품을 판매할 수 있습니다. 현재 대한민국(KRW), 일본(JPY), 인도(INR)를 지원합니다. 추가 국가 지원이 필요하면 문의해 주세요. ([contact@adrop.io](mailto:contact@adrop.io))
