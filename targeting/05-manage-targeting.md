# 타겟팅 관리하기

생성된 타겟팅과 카테고리를 수정, 삭제하고 판매 상태를 관리하는 방법을 설명합니다.

---

## 타겟팅 목록 조회

1. 좌측 메뉴에서 **타겟팅**을 클릭합니다.
2. 오디언스 타겟팅과 문맥 타겟팅이 각각 섹션으로 구분되어 표시됩니다.
3. 타겟팅 카드를 클릭하면 상세 화면으로 이동합니다.

---

## 타겟팅 노출 순서 편집

광고주가 캠페인 예약 시 타겟팅을 선택하는 화면에서 표시되는 순서를 변경합니다.

1. 좌측 메뉴에서 **타겟팅**을 클릭합니다.
2. **타겟팅 노출 순서 편집** 버튼을 클릭합니다.
3. 타겟팅을 드래그하여 순서를 변경합니다.
4. **저장** 버튼을 클릭합니다.

---

## 타겟팅 수정하기

### 타겟팅 이름 변경

1. 타겟팅 카드를 클릭하여 상세 화면으로 이동합니다.
2. 타겟팅 이름 옆 **수정** 버튼을 클릭합니다.
3. 새 이름을 입력하고 **저장**을 클릭합니다.

> ⚠️ 자동완성 타겟팅(연령대, 성별, 국가, 언어)은 이름을 변경할 수 없습니다.

### 할증률 변경

1. 타겟팅 상세 화면에서 할증률 옆 **수정** 버튼을 클릭합니다.
2. 새 할증률을 입력하고 **저장**을 클릭합니다. (0% ~ 200%)

> 💡 할증률은 기본 단가에 추가로 적용됩니다. 예: 기본 단가 10,000원, 할증률 20% → 12,000원

---

## 타겟팅 삭제하기

1. 타겟팅 상세 화면에서 **삭제** 버튼을 클릭합니다.
2. 확인 팝업에서 **삭제**를 클릭합니다.

> ⚠️ 타겟팅 내 카테고리 중 하나라도 판매 중이거나 진행 중인 캠페인에서 사용 중이면 삭제할 수 없습니다.

> 💡 타겟팅을 삭제하면 연결된 광고 상품에서 해당 타겟팅 옵션이 제거됩니다.

---

## 카테고리 관리하기

### 카테고리 상태

| 상태 | 설명 |
|------|------|
| **판매 준비 완료** | 판매 시작 가능한 상태 |
| **판매중** | 광고주가 선택 가능한 상태 |

### 카테고리 추가

1. 타겟팅 상세 화면에서 **+ 커스텀 카테고리 추가** 버튼을 클릭합니다.
2. 카테고리 이름과 조건을 설정합니다.
3. **추가** 버튼을 클릭합니다.

### 카테고리 수정

1. 수정할 카테고리를 클릭합니다.
2. **수정** 버튼을 클릭합니다.
3. 조건을 수정하고 **저장**을 클릭합니다.

> ⚠️ 판매 중이거나 진행 중인 캠페인에 사용 중인 카테고리는 수정할 수 없습니다.

### 카테고리 삭제

1. 삭제할 카테고리를 클릭합니다.
2. **삭제** 버튼을 클릭합니다.
3. 확인 팝업에서 **삭제**를 클릭합니다.

> ⚠️ 판매 중인 카테고리는 삭제할 수 없습니다. 먼저 판매를 중지하세요.

> ⚠️ 진행 중인 캠페인에 사용 중인 카테고리는 캠페인이 종료될 때까지 삭제할 수 없습니다.
