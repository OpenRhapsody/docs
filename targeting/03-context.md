# 문맥 타겟팅 만들기

문맥 타겟팅은 콘텐츠 속성을 기반으로 특정 페이지나 콘텐츠에 광고를 노출하는 기능입니다.

이 가이드에서는 **중고거래 앱**을 예시로 문맥 타겟팅을 만드는 방법을 설명합니다. 사용자가 보고 있는 상품 카테고리를 기반으로 타겟팅 카테고리를 구성합니다.

---

## 컨텍스트란?

**컨텍스트**는 사용자가 현재 보고 있는 페이지의 맥락 정보입니다. 상품 카테고리, 기사 분류, 검색 키워드 등 페이지를 구분할 수 있는 모든 데이터가 컨텍스트가 될 수 있습니다.

컨텍스트는 광고 요청 시 SDK를 통해 전달되며, 전달된 컨텍스트를 조합하여 타겟팅 카테고리를 만듭니다.

> 💡 컨텍스트 데이터의 유효기간은 28일입니다.

---

## 컨텍스트 ID 전달 방식

컨텍스트 ID는 매체사가 직접 정의합니다. 이미 서비스에서 사용 중인 콘텐츠 분류 체계가 있다면 그대로 활용할 수 있습니다. SDK에서 광고 요청 시 컨텍스트 ID를 함께 전달하면, 해당 맥락에 맞는 광고가 노출됩니다.

### 예시: 중고거래 앱

중고거래 앱에서 사용자가 보고 있는 상품 페이지의 카테고리를 컨텍스트로 전달한다고 가정합니다.

| 상품 페이지 | 컨텍스트 ID |
|-------------|-------------|
| 아이폰 15 Pro | phone |
| 맥북 프로 14인치 | laptop |
| 갤럭시 탭 S9 | tablet |
| 나이키 에어포스 | shoes |
| 구찌 토트백 | bags |
| 이케아 책상 | furniture |
| 캠핑 텐트 | outdoor |
| 유아용 카시트 | baby |

---

## 문맥 타겟팅 생성하기

### 1단계: 타겟팅 생성

1. 좌측 메뉴에서 **타겟팅**을 클릭합니다.
2. 문맥 타겟팅 섹션에서 **문맥 타겟팅 생성** 버튼을 클릭합니다.

### 2단계: 타겟팅 정보 입력

**타겟팅 이름**

광고주에게 표시될 타겟팅 이름을 입력합니다.

**할증률**

이 타겟팅을 선택했을 때 적용할 할증률을 설정합니다. (0% ~ 200%)

### 3단계: 카테고리 추가

1. **+ 커스텀 카테고리 추가** 버튼을 클릭합니다.
2. 카테고리 이름을 입력합니다.
3. 조건을 설정합니다.
   - 연산자 선택 (같음, 포함 등)
   - 컨텍스트 ID 입력
4. 필요시 **조건 추가**로 여러 컨텍스트 ID를 조합합니다.
5. **추가** 버튼을 클릭합니다.

**예시: 중고거래 앱 카테고리 구성**

컨텍스트 ID를 조합하여 다음과 같이 카테고리를 만듭니다.

| 카테고리 | 조건 |
|----------|------|
| 전자기기 | context 포함 phone, laptop, tablet |
| 패션 | context 포함 clothes, shoes, bags |
| 가구/인테리어 | context 포함 furniture, interior, home |
| 스포츠/레저 | context 포함 sports, outdoor, fitness |
| 유아동 | context 포함 baby, kids, toys |

### 4단계: 저장

**타겟팅 생성** 버튼을 클릭하여 저장합니다.

---

## 카테고리 그룹 설정

여러 카테고리를 하나로 묶어 광고주에게 제공할 수 있습니다.

1. 생성된 타겟팅 카드를 클릭합니다.
2. **카테고리 그룹 설정** 버튼을 클릭합니다.
3. 그룹 이름을 입력합니다.
4. 포함할 카테고리를 선택합니다.
5. **저장** 버튼을 클릭합니다.

> ⚠️ 그룹으로 묶인 카테고리는 개별 선택이 불가능해집니다.

**예시: 라이프스타일 그룹 만들기**

가구/인테리어와 스포츠/레저 카테고리를 묶어 "라이프스타일" 그룹을 만듭니다.

| 그룹 | 포함 카테고리 |
|------|---------------|
| 라이프스타일 | 가구/인테리어, 스포츠/레저 |

그룹 설정 후 광고주에게 표시되는 옵션:

- 전자기기
- 패션
- 유아동
- **라이프스타일** (그룹)
