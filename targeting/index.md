# 타겟팅 구성하기

광고주가 원하는 사용자에게 맞춤 광고를 노출할 수 있도록 타겟팅을 설정합니다. 잘 설계된 타겟팅은 광고 효과를 높이고, 프리미엄 가격 책정의 근거가 됩니다.

---

## 타겟팅이란?

**타겟팅**은 광고를 특정 사용자나 특정 콘텐츠에만 노출하는 기능입니다.

예를 들어, 육아용품 광고를 20~30대 부모에게만 노출하거나, 스포츠 브랜드 광고를 스포츠 기사 페이지에만 노출할 수 있습니다. 타겟팅을 통해 광고주는 원하는 고객에게 정확히 도달하고, 매체사는 보유한 데이터를 활용해 프리미엄 광고 상품을 만들 수 있습니다.

---

## 왜 타겟팅이 중요한가요?

### 데이터, 이제는 수익화할 때입니다

매체사가 보유한 데이터는 그 자체로 가치 있는 자산입니다. 회원 정보, 행동 데이터, 콘텐츠 분류 등 이미 갖고 있는 데이터를 타겟팅 상품으로 만들어 광고 수익을 높일 수 있습니다.

| 보유 데이터 | 수익화 가능성 |
|-------------|---------------|
| 회원 정보 (나이, 성별, 지역) | 인구통계 타겟팅 상품 |
| 행동 데이터 (구매 이력, 관심사) | 관심사 타겟팅 상품 |
| 금융 데이터 (소득 수준, 자산 규모) | 고소득층 타겟팅 상품 |
| 콘텐츠 분류 (카테고리, 키워드) | 문맥 타겟팅 상품 |
| 멤버십 등급 | 프리미엄 오디언스 상품 |

### 타겟팅이 만드는 차이

**타겟팅 없이 판매하면:**
- 모든 광고가 동일한 CPM으로 판매됩니다.
- 광고주는 원하는 고객에게 도달했는지 알 수 없습니다.
- 매체의 데이터 가치가 가격에 반영되지 않습니다.

**타겟팅을 활용하면:**
- 프리미엄 오디언스에 할증률을 적용해 **더 높은 단가**로 판매합니다.
- 광고주에게 **명확한 타겟팅 옵션**을 제공해 광고 효과를 높입니다.
- 보유 데이터를 **차별화된 광고 상품**으로 만들어 경쟁력을 확보합니다.

### 실제 활용 예시

> **육아 앱 A사**는 "출산 후 6개월 이내 부모" 타겟팅을 만들어 분유, 기저귀 브랜드에 **기본 CPM 대비 40% 할증**으로 판매하고 있습니다.

> **뉴스 앱 B사**는 금융, 부동산, 자동차 카테고리별 문맥 타겟팅을 제공해, 관련 업종 광고주로부터 **높은 재구매율**을 달성했습니다.

## 오디언스 타겟팅

사용자 속성 데이터를 기반으로 특정 사용자에게 광고를 노출합니다.

### 데이터 소스

**유저 프로퍼티**를 사용합니다. 국가, 언어, 성별, 나이 등 기본 필드가 제공되며, 매체사별로 필요한 커스텀 필드를 추가할 수 있습니다.

### 계층 구조

타겟팅은 **프로퍼티 → 카테고리 → 타겟팅** 순서로 만들어집니다.

**프로퍼티**는 나이, 성별, 관심사, 거주 지역, 회원 등급 같은 원천 데이터입니다.

**카테고리**는 프로퍼티에 조건을 걸어 만든 세그먼트입니다. 예를 들어 "20대 여성"은 나이 20~29, 성별 여성 조건을 조합한 카테고리입니다.

**타겟팅**은 관련 카테고리들의 집합입니다. "인구통계" 타겟팅 안에 20대 여성, 20대 남성, 30대 여성, 30대 남성 카테고리가 포함됩니다. 광고주는 타겟팅 안에서 원하는 카테고리를 선택합니다.

**카테고리 그룹**은 여러 카테고리를 하나로 합치는 기능입니다. 20대 여성과 20대 남성을 그룹으로 묶으면 "20대"라는 하나의 카테고리가 됩니다. 그룹으로 묶인 카테고리는 개별 선택이 불가능하고, 광고주는 "20대" 그룹만 선택할 수 있습니다.

### 예시: 인구통계 타겟팅

**카테고리 그룹 적용 전**

광고주 선택 옵션: 20대 여성, 20대 남성, 30대 여성, 30대 남성

**카테고리 그룹 적용 후**

- 20대 = 20대 여성 + 20대 남성
- 30대 = 30대 여성 + 30대 남성

광고주 선택 옵션: 20대, 30대

### 활용 시나리오

| 서비스 | 프로퍼티 | 카테고리 예시 |
|--------|----------|---------------|
| 육아 앱 | 출산 후 개월 수 | 신생아, 이유식기, 유아기 |
| 금융 앱 | 자산 규모 | 1억 미만, 1억 이상, 10억 이상 |
| 커머스 | 회원 등급 | 일반, 프리미엄, VIP |
| OTT | 시청 장르 | 드라마, 예능, 영화 |
| 뉴스 앱 | 관심 분야 | 정치, 경제, 스포츠, 연예 |

## 문맥 타겟팅

콘텐츠 속성을 기반으로 특정 페이지나 콘텐츠에 광고를 노출합니다.

### 데이터 소스

광고 요청 시 전달되는 **컨텍스트 ID**를 사용합니다. 컨텍스트 ID는 사용자가 현재 보고 있는 페이지의 맥락을 나타내는 값으로, 매체사가 직접 정의하여 SDK를 통해 전달합니다.

예를 들어 커머스 앱에서 사용자가 운동화 상품 페이지를 보고 있다면, "shoes" 또는 "sports_shoes" 같은 컨텍스트 ID를 전달할 수 있습니다.

### 계층 구조

타겟팅은 **컨텍스트 → 카테고리 → 타겟팅** 순서로 만들어집니다.

**컨텍스트**는 매체사가 정의한 페이지 맥락 정보입니다. 상품 카테고리, 검색 키워드, 페이지 유형 등을 자유롭게 정의할 수 있습니다.

**카테고리**는 특정 컨텍스트 ID들을 조합해서 만든 세그먼트입니다. 예를 들어 "신발" 카테고리는 shoes, sneakers, boots 등의 컨텍스트 ID를 포함하도록 구성할 수 있습니다.

**타겟팅**은 관련 카테고리들의 집합입니다. "상품 카테고리" 타겟팅 안에 신발, 의류, 가방, 액세서리 카테고리가 포함됩니다.

**카테고리 그룹**은 여러 카테고리를 하나로 합치는 기능입니다. 의류와 액세서리를 그룹으로 묶으면 "패션의류"라는 하나의 카테고리가 됩니다.

### 예시: 커머스 앱 상품 카테고리 타겟팅

**카테고리 그룹 적용 전**

광고주 선택 옵션: 신발, 의류, 가방, 액세서리

**카테고리 그룹 적용 후**

- 패션의류 = 의류 + 액세서리

광고주 선택 옵션: 신발, 가방, 패션의류

### 활용 시나리오

| 서비스 | 컨텍스트 | 카테고리 예시 |
|--------|----------|---------------|
| 커머스 | 상품 카테고리 | 신발, 의류, 가방, 액세서리 |
| 뉴스 앱 | 기사 카테고리 | 정치, 경제, 스포츠, 연예 |
| 부동산 앱 | 매물 유형 | 아파트, 오피스텔, 빌라, 원룸 |
| 레시피 앱 | 요리 종류 | 한식, 중식, 양식, 일식 |
| 여행 앱 | 여행지 | 국내, 일본, 동남아, 유럽 |

## 할증률 설정

특정 타겟에 프리미엄 가격을 적용할 수 있습니다. 광고주가 해당 타겟팅을 선택하면 기본 단가에 할증률이 추가됩니다.

| 항목 | 설명 |
|------|------|
| 설정 범위 | 0% ~ 200% |
| 적용 방식 | 기본 단가 × (1 + 할증률) |
| 누적 적용 | 복수 타겟팅 선택 시 할증률이 누적됩니다 |

**예시**: 기본 단가 10,000원, 할증률 20%인 타겟팅 선택 시 → 12,000원

## 광고 상품과 타겟팅 연결

광고 상품을 만들 때 타겟팅을 연결하는 방식을 설정합니다.

### 노출 타겟팅

광고주가 캠페인 예약 시 선택할 수 있는 타겟팅 카테고리입니다. 여기서 선택한 카테고리만 광고주에게 노출됩니다.

예: "인구통계" 타겟팅에서 20대, 30대, 40대 카테고리 중 20대와 30대만 노출 타겟팅으로 설정하면, 광고주는 20대와 30대만 선택할 수 있습니다.

### 필수 타겟팅

광고주가 반드시 선택해야 하는 타겟팅 카테고리입니다. 노출 타겟팅으로 설정한 카테고리 중에서 지정할 수 있습니다.

필수 타겟팅에는 **포함**과 **제외** 두 가지 설정이 있습니다.

| 설정 | 설명 | 예시 |
|------|------|------|
| **포함** | 광고주가 반드시 하나 이상 선택해야 하는 카테고리 | "20대" 포함 필수 → 광고주는 20대를 반드시 선택해야 캠페인 예약 가능 |
| **제외** | 광고가 노출되지 않는 카테고리 | "40대" 제외 설정 → 40대 사용자에게는 광고 노출 안 됨 |

> 포함과 제외가 중복될 경우, 제외가 우선 적용됩니다.

## 주의사항

- **오디언스와 문맥 타겟팅 동시 사용 불가**: 하나의 광고 상품에 두 유형을 함께 적용할 수 없습니다.
- **동일 유형 내 복수 타겟팅 가능**: 여러 타겟팅을 선택하면 할증률이 누적 적용됩니다.
- **카테고리 잠금**: 진행 중인 캠페인에 사용 중인 카테고리는 수정하거나 삭제할 수 없습니다.
- **데이터 업데이트 주기**: 타겟팅 데이터는 매시간 업데이트됩니다.

## FAQ

**Q. 타겟팅을 활용하면 어떤 이점이 있나요?**

A. 동일한 광고 인벤토리로 더 높은 수익을 창출할 수 있습니다. 프리미엄 오디언스에 할증률을 적용하고, 광고주에게는 명확한 타겟팅 옵션을 제공해 광고 효과를 높일 수 있습니다. 보유 데이터를 차별화된 광고 상품으로 만들어 경쟁력을 확보하세요.

**Q. 어떤 데이터를 타겟팅으로 만들면 좋을까요?**

A. "이 데이터로 광고주가 원하는 고객을 찾을 수 있는가?"를 기준으로 판단하세요. 예를 들어 육아 앱의 "출산 후 개월 수"는 분유·기저귀 브랜드에게, 금융 앱의 "자산 규모"는 프리미엄 금융 상품 광고주에게 가치 있는 타겟팅이 됩니다. 광고주 관점에서 기꺼이 할증을 지불할 만한 데이터인지 생각해 보세요.

**Q. 오디언스 타겟팅과 문맥 타겟팅 중 어떤 걸 선택해야 하나요?**

A. 보유한 데이터 유형에 따라 선택하세요. 회원 정보, 행동 데이터 등 사용자 속성 데이터가 풍부하면 오디언스 타겟팅이 적합합니다. 콘텐츠 분류 체계가 잘 갖춰져 있다면 문맥 타겟팅이 효과적입니다.

**Q. 할증률은 어떻게 설정하는 게 좋나요?**

A. 해당 오디언스의 희소성과 광고 효과를 고려하세요. 예를 들어 "고소득층", "VIP 회원"처럼 광고주에게 가치 있는 세그먼트는 높은 할증률을 적용할 수 있습니다. 시장 반응을 보며 조정하는 것도 방법입니다.
